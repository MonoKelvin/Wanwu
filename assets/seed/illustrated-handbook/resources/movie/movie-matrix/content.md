# 黑客帝国 The Matrix

《黑客帝国》（The Matrix，1999）由沃卓斯基姐妹（现莉莉与拉娜·沃卓斯基）编剧执导。程序员尼奥发现所谓现实世界竟是机器人为人类制造的模拟程序「矩阵」，在墨菲斯与崔妮蒂引导下，他逐步觉醒为预言中的救世主，对抗特工史密斯。

## 背景与历史

沃卓斯基姐妹受日本动画、赛博朋克小说与哲学课程启发，以绿色代码雨与黑色皮衣定义视觉。1999 年上映时观众对互联网仍感新鲜，「红药丸/蓝药丸」迅速进入政治与文化隐喻。武术指导袁和平参与设计打斗。

## 故事梗概

尼奥过着普通白领生活，却被神秘讯息「跟随白兔」困扰。加入地下组织后，他吞下红药丸看见真实荒漠世界：人类被机器当生物电池。他在虚拟训练中掌握功夫，却在多次任务中怀疑自己的身份。最终他在爱人「复活」与自我信念推动下，看穿矩阵代码，击败史密斯。

## 风格与影响

影片融合香港功夫片、赛博朋克、哲学（柏拉图洞穴、佛教轮回）与早期数字特效。「子弹时间」旋转镜头成为流行文化符号。绿色代码雨、黑色长风衣、墨镜造型定义了一代人的科幻审美。


## 影响与口碑

「子弹时间」技术被广告与游戏广泛模仿；续集虽评价分化，仍扩展锡安与程序人架构。2021 年《矩阵复活》尝试回应流媒体时代身份议题。

## 冷知识

- 基努·里维斯为角色大量练武，许多打斗实拍。
- 制片方曾考虑将尼奥改为普通动作英雄，导演坚持哲学深度。
- 续集《重装上阵》《矩阵革命》扩展锡安与机器城世界观。

## 延伸观赏

作品在院线、流媒体与收藏市场上持续获得新观众回顾；官方设定集、分镜集与创作者访谈适合在观赏正片后延伸阅读，有助于理解美术、配乐与叙事取舍。


## 收藏与讨论

影迷常从摄影构图、配乐动机、改编与原著差异等角度重看；蓝光收藏版、评论音轨与幕后纪录片能补充正片未展开的制作背景。学术与自媒体分析亦有助于把作品放回当时的电影工业或动画产业语境中理解。
