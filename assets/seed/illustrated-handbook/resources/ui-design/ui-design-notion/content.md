# Notion 界面

Notion 是 All-in-one workspace：块编辑器（block-based editor）统一 docs、database、wiki 与 project。界面以 sidebar 导航 + 空白 canvas 为主，slash command 插入 block，database 视图切换 table/board/calendar，形成 distinctive productivity UI 语言。

![Notion Logo](https://upload.wikimedia.org/wikipedia/commons/thumb/e/e9/Notion-logo.svg/640px-Notion-logo.svg.png)

## 设计师与品牌

Notion Labs 2016 年 San Francisco；创始人 Ivan Zhao、Simon Last。2021 年估值高峰；面向 team 与 enterprise SSO、audit log。竞品含 Confluence、Coda、飞书文档、语雀。Notion AI 2023 嵌入 writing 与 Q&A。

## 设计亮点

Block model：paragraph、heading、todo、toggle、callout、code 均为 block，drag handle 重排。Database：property types（select、relation、rollup、formula）+ 多 view。Inline database 嵌入 page。Typography 克制：system-ui stack、灰阶 hierarchy。Sidebar nested page tree；breadcrumb 与 backlink 支持 graph。Mobile 简化 editing；quick capture widget。Dark mode 完整 token。

## 使用体验

`/命令` 快速插入；@mention 人与 page。Template gallery 复制 workflow。Relation 链 project 与 doc 减少 duplicate。Performance 大 database 需 filter index；offline 有限。Export markdown/HTML/PDF。Team：permission page-level、guest、comment。Learning curve：concept 简单但 advanced database 需 tutorial。

## 文化影响

Notion 定义「 aesthetic minimal productivity」：大学生、startup、indie hacker wiki 标配。YouTube「第二大脑」「PARA」方法论依赖 Notion template 传播。UI 影响 Slack canvas、Linear docs 等产品的 block 化趋势。中国用户常用 Notion 国际版或迁移至飞书/语雀，但 block + database 范式已普及。
