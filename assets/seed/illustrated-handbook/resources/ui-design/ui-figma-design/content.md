# Figma 设计

Figma 是 browser-native 协作界面设计工具，支持实时 multiplayer editing、component library 与 handoff。作为 2020 年代 UI/UX 工作流中心，它连接 research、visual design、prototype 与 Dev Mode 代码交付，替代传统 Sketch + Zeplin 链路。

![Figma 界面 Logo](https://upload.wikimedia.org/wikipedia/commons/thumb/3/33/Figma-logo.svg/640px-Figma-logo.svg.png)

## 设计师与品牌

Figma Inc.，CEO Dylan Field；2024 UI3 简化 toolbar 与 panel。Plugins 生态含 Content Reel、Iconify、Anima。Enterprise：SCIM、audit、design system analytics。Education 免费 tier 培养用户习惯。Adobe 收购失败后独立 IPO 预期持续。

## 设计亮点

Multiplayer cursor 降低 review 邮件往返。Component properties：boolean、instance swap、text prop。Auto Layout 对应 CSS flex/grid 心智。Variables 2.0 支持 scoping 与 modes（light/dark/brand）。Prototype interactions 可 user test 前验证 flow。FigJam sticky、diagram 与 design file 互通。AI features：rename layers、generate content（功能随版本更新）。

## 使用体验

快捷键：R frame、A text、K scale；option-drag duplicate。Library publish 需 manage permission。Branching explore rebrand 安全。Dev inspect 复制 iOS/Android/CSS。Performance：hide other pages、use component not duplicate group。Handoff ritual：「Figma link in ticket」成为 sprint 标准。

## 文化影响

Figma 使 junior designer 也能维护 system library；YouTube「design in public」直播 Figma。Community 文件传播 dashboard clone 学习。与 engineering 边界模糊：design engineer 直接在 Figma Code Connect 映射 React。中国即时设计、Pixso 等对标其协作模型。
