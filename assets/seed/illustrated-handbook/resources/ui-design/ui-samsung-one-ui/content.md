# 三星 One UI

Samsung One UI 是三星 Galaxy 设备上的 Android 定制界面，2018 取代 Samsung Experience，强调 reachability（单手操作）、focus block 与大图标 spacing。One UI 6/7 基于 Android 14+，融合 Material You 同时保留 Samsung icon、Settings 层级与 Good Lock 深度定制。

![Samsung Galaxy One UI 示意](https://upload.wikimedia.org/wikipedia/commons/thumb/2/24/Samsung_Logo.svg/640px-Samsung_Logo.svg.png)

## 设计师与品牌

Samsung MX（Mobile eXperience）设计中心首尔；与 Google AOSP 协作适配。Good Lock、Theme Park、Keys Cafe 等模块供 power user 改 UI。Galaxy Z Fold/Flip 折叠屏有专属 taskbar、flex mode layout。

## 设计亮点

Screen 分 top viewing（内容）与 bottom interaction（thumb zone）。Quick settings 大 tile；Settings 搜索 prominent。Icon 圆角 squircle 与 tint 与 stock Android  differentiate。Always On Display、Edge panels 增 glance info。One UI Watch 与 Wear OS 整合 Galaxy Watch。DeX 桌面模式 windowing 近似 desktop UX。

## 使用体验

用户：navigation gestures 可改三段式；bixby 可 hide。Developer：test app on One UI 安全文件夹、multi window。Fold 适配：cover vs inner display resource qualifier。Update 节奏：旗舰一年 major，mid-range 延迟。Good Lock 可改 lock screen 无 root。

## 文化影响

One UI 与 MIUI、ColorOS 代表 Android 碎片化 UX；Material Pure 爱好者 debate customization vs consistency。折叠屏 UX 三星先行，影响 Google large screen guidelines。全球 shipment 第一厂商 UI 习惯影响 third-party app 默认 layout 测试优先级。
