# Figma 设计稿

Figma 设计稿（Design File）是 UI/UX 协作的核心交付物：Frame 表页面、Component 与 Variant 管理设计系统，Auto Layout 模拟 flex，Prototype 连接交互。Dev Mode 与 inspect 面板让设计与工程共享 spacing、token 与 export asset。

![Figma Logo](https://upload.wikimedia.org/wikipedia/commons/thumb/3/33/Figma-logo.svg/640px-Figma-logo.svg.png)

## 设计师与品牌

Figma Inc. 2016 年发布 browser-first 设计工具；2022 年 Adobe 收购案终止后独立发展。2024 年推出 Figma Config 与 UI3 界面 refresh。竞品含 Sketch、Adobe XD（式微）、Penpot 开源。FigJam 承担 whiteboard；Figma Slides 扩展 presentation。

## 设计亮点

Components + Variants：property（size、state）组合实例。Auto Layout：padding、gap、fill/hug 对应 CSS flex。Variables：color、number、string 绑定 light/dark mode。Styles  legacy 向 variables 迁移。Prototype：smart animate、overlay、scroll behavior。Dev Mode：CSS/iOS/Android snippet、compare changes。Library 发布 team design system；branching 支持 explore 不破坏 main file。

## 使用体验

设计师：constraints + layout grid 适配 responsive；命名 layer 便于 handoff。开发：MCP/Dev Mode API 拉 token；export SVG/PNG @1x/@2x。PM：comment、annotation、status sticker。大 file 用 page 分区；performance 问题可 flatten 或 split library。Organization 管理 seat、plugin 与 SSO。

## 文化影响

Figma 取代 Sketch 成为多数 tech 公司标准，「Figma file link」代替 PSD handoff。Community file 传播 dashboard、mobile kit 审美。Config 大会发布 Variables、Dev Mode 影响 industry workflow。中国团队广泛使用 Figma 或即时设计、MasterGo 等本土协作工具对照其功能集。
