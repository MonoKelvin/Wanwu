# 缅甸猫

缅甸猫（Burmese）源自东南亚古老家猫类型，被毛短密呈缎光、眼色金黄，性格社交亲人。现代品种分「美国型」（更紧凑圆润）与「欧洲型」（略修长），均强调温润毛色与「会说话」的陪伴特质。

## 背景与历史

1930 年代，一只来自缅甸的深褐色雌猫「Wong Mau」抵达美国旧金山，经与暹罗猫杂交后确立缅甸猫血统；毛色基因（Burma 等位）与暹罗重点色不同，呈全身均匀貂色至巧克力色。英国于 1940–50 年代自东南亚再引入血系，形成与美式略有差异的标准。今日 CFA、GCCF、FIFe 均登记，是东方短毛家族中最「圆润」的一支。

时间线与地域背景有助于理解它为何在特定年代走红，以及今日在收藏、实用或文化象征中的位置。


## 外观特征

- **体型**：美式紧凑肌肉型，公猫 4–5.5 kg；欧洲型略长但仍是中等体型。
- **头型**：圆头圆眼，耳大前倾，吻部短，呈现「娃娃脸」。
- **被毛**：短、贴身、光泽如缎，几乎无底层绒毛感；认可貂色、香槟色、蓝色、铂金色等。
- **眼色**：金黄至琥珀，明亮有神，是品种灵魂特征。
- **识别要点**：缎光短毛 + 金眼 + 圆润体型；与新加坡猫相比体型更大、毛色更深。

## 性格与气质

缅甸猫极度亲人，有「跟脚猫」传统，常随主人从一个房间走到另一个。爱发声交流，音调低沉不刺耳。独处易焦虑，适合有人在家或养伴猫。对儿童、犬只通常友好，游戏欲强，老年仍爱互动。

## 饲养与护理

- **梳毛**：每周 1 次软刷即可，被毛易打理。
- **陪伴**：需每日互动，否则可能出现过度喵叫或破坏行为。
- **健康**：关注颅面缺陷（美式血系历史问题，正规猫舍已大幅改善）、糖尿病与肥胖。
- **饮食**：高代谢倾向，控制碳水与零食，维持肌肉而非脂肪。

## 文化与冷知识

- Wong Mau 最初被误认为暹罗猫，遗传研究才揭示其独立毛色基因，是猫遗传学史上的经典案例。
- 在英语圈，缅甸猫有「Velcro cat」（魔术贴猫）昵称，形容其黏人程度。
- 缅甸猫与孟买猫、东奇尼猫共享部分东南亚血统背景，育种史相互交织。
