# 北海道猫（地方型）

北海道猫指日本**北海道**寒冷地区长期自然繁衍形成的地方型家猫，以厚实被毛、结实体型与耐寒适应著称。它并非单一国际登记品种，而是与本土短毛、混血家猫及可能受日本短尾猫影响的地理种群并列讨论的地方类型——类似中国「华北田园猫」的概念。

## 背景与历史

北海道纬度较高、冬季漫长，家猫随移民、渔业与贸易进入后，在户外与半户外环境中自然选育：被毛更密、皮下脂肪与骨量略增、活动模式适应雪季。日本国内对「北海道猫」并无像美国 CFA 那样的独立标准，但在宠物文化与地方宣传中常作为**寒冷地家猫**的代表形象。

与**日本短尾猫（Japanese Bobtail）**相比，北海道型未必尾短，更强调「厚毛 + 壮实」；与**挪威森林猫**等长毛品种无直接血缘关系，是环境塑造表型而非人工品系。与**冲绳猫**等日本其他地区类型一样，体现群岛家猫的地域多样性。

## 外观特征

- **被毛**：双层毛，冬毛尤其浓密；毛长从短毛到半长毛均有，以能御寒为实际标准。
- **体型**：中等到偏大，胸宽、四肢结实，体重感强于关西等温暖地区瘦型家猫。
- **毛色**：虎斑、玳瑁、白、黑、三毛等各色均有，无固定配色要求。
- **耳型**：略圆，耳中等，耳尖有簇毛个体可见（非品种必具特征）。
- **尾**：多数为正常长度；短尾个体存在但不作为类型定义。

## 性格与气质

北海道猫整体气质偏**独立、机警、适应户外能力较强**——与寒冷地区资源与半散养历史相关。家养个体可非常亲人，尤其自幼室内社会化者；部分保留对门窗、阳台与低温环境的兴趣。与纯种室内猫相比，更需关注丰容与安全户外（如封闭阳台）。

## 饲养与护理

- **换毛**：冬毛转夏毛期间掉毛量显著，需每日梳毛与吸尘。
- **温度**：室内无需刻意低温，但避免夏季空调直吹；提供温暖睡垫即可。
- **体重**：壮实体型易与肥胖混淆，定期称重、触诊肋骨判断体况。
- **活动**：提供攀爬与狩猎游戏；若接触室外，需芯片、疫苗与 TNR 意识。
- **健康**：无品种特异遗传病清单，按家猫常规体检即可。

## 文化与冷知识

- 北海道旅游素材中常出现「雪地里的大毛猫」，多为地方家猫或混血，成为地域视觉符号。
- 日本「地方猫」讨论常包括东京猫、大阪猫、冲绳猫等——多为半玩笑的地理性格刻板印象，科学上证据有限。
- 厚毛家猫在东亚北方（含中国东北田园猫）有类似表型，属**平行适应**而非同源品种。
- 若在日本领养，多数仍为 mix，勿与品种猫舍混为一谈。
