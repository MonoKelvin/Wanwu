# iPhone 设计

iPhone 自 2007 年由 Apple 发布，重新定义智能手机的工业设计与交互范式：正面单块玻璃、极简按键、一体化铝/不锈钢中框与严格的 R 角、天线带语言。历代演进在「更薄、更大屏、更少孔」与耐用、影像模组增大之间平衡，成为全球移动硬件美学的参照系。

![iPhone 产品演进](https://upload.wikimedia.org/wikipedia/commons/thumb/f/fa/IPhone_6_Silver.png/320px-IPhone_6_Silver.png)

## 设计师与品牌

Apple 设计团队长期由 Jony Ive 领导至 2019 年，后由 Evans Hankey、Alan Dye 等接任。iPhone 与 iOS Human Interface Guidelines 协同，硬件圆角与 UI 圆角一致。供应链由中国富士康等组装，工业设计在加州 Cupertino 完成。Pro 系列使用钛金属、Ceramic Shield 等新材料叙事。

## 设计亮点

iPhone 4 不锈钢中框 + 玻璃背板建立现代比例；iPhone X 去 Home 键、刘海 Face ID 改变正面 purity。Dynamic Island（14 Pro 起）将前置传感器区功能化。按钮布局、静音拨片与 MagSafe 磁吸环形成家族识别。相机 bump 从单摄到三摄 Pro 阵列，影响全手机行业背板语言。包装减塑、无充电器策略亦属产品设计延伸。

## 使用体验

日常握持感、单手可达性与重量分布每代微调；Pro Max 大屏适合影音与摄影， mini 线（已停）曾服务小屏偏好。钛金属 Pro 减轻重量但易留指纹。MagSafe 生态含充电器、卡包与三脚架。维修性较 Android 部分机型低，官方与授权维修是主流路径。颜色年度更新影响二手保值率。

## 文化影响

iPhone 轮廓被无数配件、UI mockup 与 Android 早期「致敬」引用。它是 2000 年代后工业设计史上销量最大、文化渗透最深单品之一。中国供应链与本土品牌（华为、小米、OPPO）在材质、摄像模组与折叠形态上与之对话或差异化。设计教育常以其讲解「品牌系统 + 软硬件一体 + 年度迭代节奏」。
