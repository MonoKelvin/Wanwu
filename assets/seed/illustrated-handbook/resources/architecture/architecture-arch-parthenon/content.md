# 帕特农神庙

帕特农神庙，是 雅典卫城 最重要的主体建筑，又译“巴特农神庙”。

地标建筑既是结构工程，也是城市记忆与旅游符号的载体。

![帕特农神庙](https://upload.wikimedia.org/wikipedia/commons/thumb/d/da/The_Parthenon_in_Athens.jpg/960px-The_Parthenon_in_Athens.jpg)

## 结构与建造

- **中文名**：帕特农神庙
- **外文名**：Parthenon Temple
- **地理位置**：希腊雅典卫城的古城堡中心
- **所处时代**：公元前447年
- **占地面积**：2137 m²

- 地标建筑常融合结构创新与文化符号，参观动线影响体验节奏。

## 影响与冷知识

- 结构暴露或装饰繁复，常反映建造年代的技术自信与文化取向。
- 相关标签：希腊雅典卫城的古城堡中心, Parthenon Temple, 帕特农神庙。
