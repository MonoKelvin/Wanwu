# 金门大桥

金门大桥（Golden Gate Bridge），又称“金门海峡大桥”，是 美国 境内连接 旧金山 市区和北部的 马林县 的跨海通道，位于 金门海峡 之上，是美国旧金山市的主要象征。

地标建筑既是结构工程，也是城市记忆与旅游符号的载体。


![金门大桥](https://upload.wikimedia.org/wikipedia/commons/thumb/b/bf/Golden_Gate_Bridge_as_seen_from_Battery_East.jpg/960px-Golden_Gate_Bridge_as_seen_from_Battery_East.jpg)



## 结构与建造

- **中文名**：金门大桥
- **外文名**：Golden Gate Bridge
- **始建日期**：1933年1月5日
- **投用日期**：1937年5月28日
- **所属国家**：美国
- **类 型**：悬索桥 、 公路桥 、 特大桥 、跨海桥

- 地标建筑常融合结构创新与文化符号，参观动线影响体验节奏。

## 影响与冷知识
