# 佩特拉古城

佩特拉（Petra），又称“玫瑰城”或“岩石城”（希腊语译名），《 圣经·旧约 》称其为Sela，位于 约旦 安曼 西南260公里的穆萨山谷，面积26,171公顷，属沙漠气候，因赤红色砂岩在阳光下呈玫瑰色调得名 。

地标建筑既是结构工程，也是城市记忆与旅游符号的载体。

![佩特拉古城](https://upload.wikimedia.org/wikipedia/commons/thumb/e/e8/Al_Deir_Petra.JPG/960px-Al_Deir_Petra.JPG)

## 结构与建造

- **中文名**：佩特拉
- **外文名**：Petra
- **别 名**：石城、玫瑰城
- **性 质**：约旦的一座古城
- **位 置**：约旦 安曼 南260公里处
- **所属国家**：约旦

- 地标建筑常融合结构创新与文化符号，参观动线影响体验节奏。
