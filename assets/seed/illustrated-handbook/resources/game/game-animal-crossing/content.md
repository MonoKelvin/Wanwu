# 集合啦！动物森友会

《集合啦！动物森友会》是任天堂 Switch 上的生活模拟游戏，玩家作为无人岛移民代表，与动物村民共建岛屿、布置家居、参与钓鱼捕虫与季节活动。

![动物森友会游戏画面](https://upload.wikimedia.org/wikipedia/en/thumb/1/1f/Animal_Crossing_New_Horizons.png/220px-Animal_Crossing_New_Horizons.png)

## 玩法特色

- **实时时钟**：游戏内日期与现实同步，北半球/南半球季节可切换。
- **DIY 与地形编辑**：解锁工具后可改造河流悬崖，自定义岛屿景观。
- **联机访问**：好友可互访岛屿交易水果、欣赏设计。

## 社会现象

2020 年疫情期间成为现象级社交游戏，带动 Switch 销量与玩家自制服装设计分享文化。

## 冷知识

- 每个存档的岛屿水果种类固定，常与好友「互访换果」。
- 大头菜（ turnip ）炒股玩法源自日本「カブ」谐音梗。
- 博物馆由玩家捐赠鱼类、化石与艺术品逐步填满。
