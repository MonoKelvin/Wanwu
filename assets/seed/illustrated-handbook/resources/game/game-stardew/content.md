# 星露谷物语（Stardew Valley）

《星露谷物语》是 ConcernedApe（Eric Barone）一人开发的农场生活模拟游戏。玩家继承祖父的旧农场，在星露谷种植作物、饲养动物、与村民社交、探索矿洞与节日活动。

![星露谷物语农场场景](https://upload.wikimedia.org/wikipedia/en/thumb/f/fd/Stardew_Valley_cover.jpg/220px-Stardew_Valley_cover.jpg)

## 核心循环

- **四季农耕**：春种夏长，作物与鱼种随季节变化。
- **社区中心**：收集包完成献祭或选择 JoJo 超市路线，影响村庄发展。
- **关系系统**：送礼、对话与节日事件可缔结婚姻与友谊。

## 影响

以《牧场物语》为灵感，以极低团队规模证明独立农场游戏的市场潜力，并带动后续《珊瑚岛》等同类型作品热潮。

## 冷知识

- 开发者 Barone 独自完成编程、美术与音乐的大部分工作，历时约四年。
- 矿井深处有骷髅与史莱姆战斗，与田园表层形成反差。
- 社区普遍推荐第一年完成献祭路线以解锁巴士与温室。
