# 巫师 3：狂猎

《巫师 3：狂猎》是 CD Projekt RED 开发的开放世界 RPG，改编自安德烈·萨普科夫斯基的猎魔人小说。玩家扮演利维亚的杰洛特，在战争纷乱的大陆追寻养女希里，对抗狂猎军团。

![巫师3 游戏封面](https://upload.wikimedia.org/wikipedia/en/thumb/0/0c/Witcher_3_cover_art.jpg/220px-Witcher_3_cover_art.jpg)


从读者角度，把「标准书描述」与「真实饲养体验」对照着看，更容易判断自己是否适合该主题：时间投入、预算、空间与家庭成员（老人、幼儿、其他宠物）都会改变答案。以下内容在常识基础上稍作延展，便于形成 3–5 分钟可读完的完整印象。

## 特色系统

- **分支叙事**：主线与支线选择影响结局与 NPC 命运，如史凯利杰群岛王位、尼弗迦德战争走向。
- **炼金与法印**：剑油、炸弹与伊格尼法印构成战斗准备循环。
- **昆特牌**：内置卡牌小游戏成为社区文化符号。

## 评价与 DLC

2015 年发售获广泛赞誉；资料片《石之心》《血与酒》被视作行业 DLC 标杆，后者新增陶森特公国地图。

## 冷知识

- 开发团队曾公开感谢中国玩家对正版的支持。
- 马匹「萝卜」因 AI 卡地形曾引发大量玩家梗图。
- 游戏内大量怪物设计参考斯拉夫与北欧民间传说。
