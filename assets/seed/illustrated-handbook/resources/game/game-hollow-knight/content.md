# 空洞骑士（Hollow Knight）

《空洞骑士》是澳大利亚 Team Cherry 开发的类银河战士恶魔城独立游戏。小骑士深入圣巢遗迹，以 nail 剑战斗、解锁能力、对抗感染与古老秘密。

![空洞骑士宣传图](https://upload.wikimedia.org/wikipedia/en/thumb/9/9f/Hollow_Knight_cover_art.jpg/220px-Hollow_Knight_cover_art.jpg)

## 玩法特点

- **非线性地图**：获得冲刺、超级冲刺、暗影披风等能力后回溯探索新区域。
- **护符系统**：装备槽有限，需在攻击、防御与探索增益间取舍构筑。
- **高难度 Boss**：神居 DLC 提供额外挑战与剧情补完。

## 独立游戏标杆

手绘像素美术与 Christopher Larkin 配乐广受好评，以 modest 预算证明小团队可做出世界级 Metroidvania 作品。

## 冷知识

- 续作《空洞骑士：丝之歌》以大黄蜂为主角，开发周期较长成为玩家梗。
- 地图商人 Cornifer 的哼唱声成为社区标志性音效。
- 游戏世界观通过环境叙事与零散文本逐步揭示，几乎无过场动画解说。
