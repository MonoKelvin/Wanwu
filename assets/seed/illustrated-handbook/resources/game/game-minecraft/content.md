# 我的世界（Minecraft）

《我的世界》是 Mojang 开发的沙盒建造游戏，玩家可在 procedurally 生成的方块世界中采集资源、合成工具、建造建筑，或对抗怪物、探索下界与末地。创始人 Markus Persson（Notch）于 2009 年发布早期版本，2014 年被微软收购后持续更新。

![Minecraft 方块世界](https://upload.wikimedia.org/wikipedia/en/thumb/5/51/Minecraft_cover.png/220px-Minecraft_cover.png)

## 核心玩法

- **生存模式**：管理饥饿与生命值，昼夜循环中抵御僵尸、苦力怕等敌对生物。
- **创造模式**：无限资源与飞行，适合建筑与红石机关实验。
- **多人联机**：支持局域网和服务器，社区模组（Mod）生态极为丰富。

## 文化与影响

方块美学与低门槛创意使其成为全球销量最高的游戏之一，广泛用于教育编程（如 Minecraft Education）与流媒体内容创作。

## 冷知识

- 游戏内一天约等于现实 20 分钟。
- 苦力怕灵感来自一次猪模型代码错误导致的膨胀效果。
- 末影人不会直视玩家眼睛，这一设定成为网络梗文化来源之一。
