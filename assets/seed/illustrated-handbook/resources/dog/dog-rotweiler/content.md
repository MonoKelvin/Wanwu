# 罗威纳犬

罗威纳犬（Rottweiler）源自德国南部罗特威尔一带，是古老的工作犬与护卫犬代表。黑褐双色、体格敦实、咬合力强，在警用、护卫与家庭守护场景中广为人知；经系统社会化与正向训练后，对家人温和可靠，但不宜由缺乏经验者随意饲养。

## 背景与历史

其祖先可追溯到罗马军团随军牧牛犬，在中世纪罗特威尔肉类贸易重镇担任赶牛、护卫钱袋与财产的角色。19 世纪铁路兴起后一度数量锐减，20 世纪初德国爱好者重建血系，并制定体型与性格标准。今日 FCI、AKC 等均承认该品种，强调稳定神经类型与可训练性，反对仅以「凶猛」为卖点的极端选育。

## 外观特征

- **体型**：大型犬，公犬肩高约 61–69 cm，体重常见 50–60 kg；结构紧凑、胸深、后躯有力。
- **头型**：颅顶宽、吻部有力，耳部自然下垂或按标准裁耳（部分地区禁止裁耳）。
- **被毛**：短而贴身，主色黑、眉弓/颊部/四肢及胸前为栗褐至棕褐标记。
- **识别要点**：黑褐配色 + 壮实轮廓 + 沉稳步态；与杜宾相比更厚重，与拳师犬相比被毛更短。

## 性格与气质

对家人忠诚、保护欲强，学习能力强，适合服从、护卫、搜救等工作。对陌生人常保持警觉，需幼年期广泛接触人、犬与环境。运动与心智刺激不足时易出现破坏或过度吠叫。与儿童相处需监督，因其力量大、兴奋时冲撞力强。

## 饲养与护理

- **运动**：每日至少 1–2 小时高强度活动（跑步、拖拽、嗅闻训练等）。
- **训练**：早期社会化必不可少；建议专业服从课程，避免粗暴惩罚引发防御攻击。
- **健康**：关注髋关节发育不良、肘关节病、心脏问题及某些肿瘤高发倾向；控制体重减轻关节负担。
- **法规**：部分城市对「烈性犬」有登记或禁养规定，饲养前须了解当地条例。

## 文化与冷知识

- 名称来自德国城镇 Rottweil，曾被称为「罗特威尔屠夫犬」。
- 历史上曾用颈圈挂钱袋，传说其守财可靠，成为贸易文化符号。
- 影视与新闻中的负面个案常源于饲养不当，品种俱乐部长期推动责任饲养教育。
