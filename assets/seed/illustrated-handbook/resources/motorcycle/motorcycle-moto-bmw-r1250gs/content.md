# BMW R 1250 GS

BMW R 1250 GS 是 **冒险旅行（ADV）** 品类的全球标杆：水平对置「拳击手」双缸、长行程悬挂与庞大原厂改装目录，使它在公路、碎石路与长途露营场景中成为「两轮 SUV」的代名词，销量长期领跑欧洲大型 ADV 市场。

![BMW R 1250 GS](https://upload.wikimedia.org/wikipedia/commons/thumb/9/9e/BMW_R1200GS_LC.jpg/640px-BMW_R1200GS_LC.jpg)

## 背景与历史

GS（Gelände/Straße，越野/公路）系列始于 **1980 年 R 80 G/S**，以 Paris-Dakar 拉力赛胜利建立 ADV 基因。2013 年 **R 1200 GS LC**（Liquid Cooled）引入水冷拳击手发动机；**2018 年 R 1250 GS** 将排量提升至 1254 cc，加入 ShiftCam 可变气门正时，2019 年起标配 ABS Pro 与 DTC 牵引力控制。

Rallye、Adventure、HP 等套装面向不同越野比例；**R 1250 GS Adventure** 加大油箱与保护件。GS 与 GSA 在欧洲注册量中占据 BMW 摩托业务核心，亦深刻影响 KTM 1290 Super Adventure、杜卡迪 Multistrada 等产品定位。

## 工程与设计

**1254 cc 水平对置双缸** 约 **136 hp / 143 N·m**，ShiftCam 在低转速改善扭矩、高转速扩展功率。轴传动降低链条维护，是 GS 长途可靠性标签之一。Telelever 前桥与 Paralever 后桥控制加速 squat 与制动 dive，整备质量约 249 kg（标准 GS）。

7 英寸 TFT、动态 ESA 电子悬挂、弯道 ABS Pro 与多种 Riding Mode 构成电子套件。风挡高度可调，原厂 panniers、crash bar、LED 辅助灯形成完整 touring ecosystem。

## 性能与体验

GS 的强项是 **全能与可控**：公路巡航稳定，拳击手低重心利于站立越野；悬挂行程吸收碎石与 pothole。ShiftCam 使低速跟车更顺，高速超车仍有储备。Adventure 版本车重与座高更高，矮个骑手需考虑 **820 mm+ 座高**。

非专业越野场景下，GS 足够应对 gravel 与浅沙；极限 mud 仍逊于专用 enduro。维护成本高于日系 650 ADV，但 BMW 经销商网络与二手保值在欧洲较强。

## 文化地位与影响

R 1250 GS 定义了 **「GS 文化」**：Alpe 穿越、Trans-America 与 GS Trophy 车主活动构成全球社群。Ewan McGregor《Long Way》系列多次使用 BMW GS，强化大众对 ADV 长途的想象。

对中国读者，GS 是 **高端 ADV 参考系**——讨论金翼、K1600 之外，「要不要拳击手发动机与轴传动」往往从 GS 开始。

首次选购大型 ADV 者建议在经销商处对比 R 1250 GS 与 GSA 座高与重量，必要时选 lowering kit 或 Rally 低座版本。
