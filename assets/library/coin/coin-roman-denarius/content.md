# 罗马银币第纳尔

## 概述

第纳尔是起源于罗马帝国银币Denarius的货币单位，其名称在西班牙语和葡萄牙语中保留词源特征。Denarius继承Solidus金币后，其涵义演变为指代金币 。现流通于伊拉克、利比亚、突尼斯、阿尔及利亚等国。阿尔及利亚第纳尔由中央银行发行，纸币面值包括5至1000第纳尔，硬币面值包括1分、2分、5分、10分、20分、50分及1第纳尔、5第纳尔、10第纳尔 。
