# 熊猫金币

熊猫金币是 中国人民银行 自1982年开始发行的一款成系列发行的金币。在迄今为止的40年时间里，以 中国国宝 —— 熊猫 为主图的熊猫金币作为中国人民银行发行的普制金币。 熊猫金币以其独特的选题、 奇美 的图案、 栩栩如生 的画面处理、精湛的铸造工艺、标准的成色、齐全的规格、超群的质量等特点赢得了国内外集藏界的一致 赞誉 和好评。 中国人民银行定于2017年10月30日起陆续发行2018版熊猫金银纪念币一套12枚，其中熊猫普制金银纪念币6枚（2017年10月30日发行）、熊猫精制金银纪念币6枚（2017年12月8日发行），均为中华人民共和国法定货币。 [1] 中国人民银行定于2019年10月30日发行2020版熊猫金银纪念币一套12枚，其中熊猫普制金银纪念币6枚、熊猫精制金银纪念币6枚，均为中华人民共和国法定货币。 [2] 2026年5月9日“中国金币”发布关于熊猫金币投资消费有关事宜的风险提示。 [15]

从1982年熊猫金币试销开始，中国熊猫金币在世界开始闻名，并同时成为中国金币的品牌产品。一览世界普制金币概貌：南非的福格林金币是1970年发行；加拿大的枫叶金币是1979年发行；美国的鹰洋金币是1986年发行；英国不列颠金币也是1986年发行；澳大利亚的鸿运金币是1987年发行；奥地利的音乐金币则是1989年发行，还有墨西哥的自由女神金币、新加坡的狮子金币等等，由此可见，中国熊猫金币1982年发行应当说是靠前的。截至2013年，中国熊猫金币共发行32年份的不同版别。

30余年来，中国熊猫金币以其独特的立意主题、精美的图案设计、精湛的铸造工艺、标准的成色、齐全的规格等在世界币林中独树一帜，成为世界公认的五大投资币之一，屡获国际大奖，赢得了国内外一致赞誉。

作出发行以熊猫为题材的普制金币的决定后，熊猫金币研制、开发、生产的任务首先就落到了中国人民银行印制局局长杨秉超身上。这位新四军的老战士，我国印钞造币企业的出色领导人，所面临的现实情况是，造币企业以生产铝合金分币为主，其设备不具备生产金币的条件，虽然已积累了生产《建国30周年》等纪念金币的经验，但前几套金币产量较小，又非高纯成色，而生产熊猫金币生产量大，纯度和成色要求高，需要先进的熔、轧、冲、压设备，还需要精确的称量设备——这些设备在加拿大国家造币厂已不是问题，而中国却缺少这些先进的生产设备。因此，熊猫金币的发行面对的是系统工程，在改革开放初期没有大量资金投入，发行时间又极为紧迫的情况下，要顺利完成任务，困难可想而知。

印制局领导首先组织科技处、生产处、经销处进行工作分工，然后发动造币厂，依靠强大的组织力量，号召技术人员和车间工人自力更生，群策群力，从熔化、轧片、冲饼、滚边、清洗、烘干、压印，到最后称量，对每一个工序进行革新、改造，发挥大家的积极性和创造性。这一期间，技术人员与车间工人们自己动手创制、革新了很多设备，没有金币坯饼抛光机，就研制多台半手工的抛光机，工人的手指都磨破了，但保证了抛光工艺的质量，仍用这些机器生产的产品外观和质量仍很完美；没有精准的现代化称量工具，就铸造小锉来锉磨坯饼，确保金坯饼的重量精准：没有无尘车间的条件，就因陋就简，保持清洁。熊猫金币在我国的试制和生产，是我国技术人员和工人同志们自力更生艰苦奋斗优良传统的生动写照，也是改革开放初期全国人民意气风发加快社会主义建设的一个缩影。

1982年发行的第一枚熊猫币在改革开放的前提下接轨国际惯例，既是钱币文化交流的产物，也承担着为中国出口创汇、增加外汇储备服务的部分功能。早期的熊猫币以出口为主。也正是因为改革开放的政策，中国开始在国际舞台上展示自己的形象，中国熊猫币也开始在国际钱币界崭露头角，出口海外，赢得荣誉。

中国的改革开放没有模式可以借鉴，只能摸着石头过河。中国熊猫币同样没有模式可以借鉴，虽然国际上投资金币并非空白，甚至已有成功先例，如南非福格林金币，但是它成色仅为91.6%，且每年的图案不变，这和中国人对黄金重纯重意是不同的。通过对他国经验的吸收借鉴，中国熊猫金币匠心独运，创出了有中国特色的熊猫币发展之路，我们选其节点研究背景，探索发展之路，也期望在不断地摸索中完善。

5周年：首次发行，规模小，发行量少。1987年第一次发行的“中国熊猫金币发行5周年”纪念银币的发行量和规格都较小，仅发行了2枚银币，分别为5盎司和1盎司。这是由当时实际情况决定的，毕竟熊猫币的发行时间还比较短，发行量、规格和所用材质都比较谨慎，既是5周年阶段的庆祝纪念，也是试探市场的反应。

10周年：规格大，发行量小。1991年熊猫金币发行10周年纪念的时候，熊猫金币已经有10年的经验，市场运营也取得了良好的进展。1991年 “中国熊猫金币发行10周年”中的大规格5公斤金币，发行量仅为10枚。正是这枚发行量最少的金币，创造了中国现代金银币单枚币最高价格的神话。2012年这枚熊猫金币发行10周年5公斤金币的市场价格高达8位数，当年的些微坎坷反倒使它价值爆发，正验证了中国的那句俗语“是金子总要发光的”。这枚5公斤金币巧妙地把10年熊猫币的不同图案汇集设计在一幅“戏竹图”中，构思独特，又极契合主题，只只熊猫仿佛聚在一起叙旧、畅谈，似是在开一个圆桌会议，各抒己见、各展风姿，共同感受这10年来熊猫币走过的辉煌历程，共谋今后发展之路。

在经历了发行的短暂平淡和游资的爆炒之后，熊猫币一如熊猫一样，在环境的波折动荡中坚守着自己最初的从容，在风波过后能稳健复苏，欣欣向荣，生生不息。

15周年：图案设计逐步成熟，坚持做好自己。1996年熊猫币发行15周年系列金银币之时，中国的金银币市场并不如熊猫币刚起步时走得那么顺利，但是熊猫币在不利环境下，仍然坚持做好自己，没有因为客观条件降低自己的品质。“中国熊猫金币发行15周年”系列金银币中的10元和25元规格的纪念金币采用了同一个图案，即1996年当年熊猫金币的图案。一只可爱的小熊猫攀爬到树干上，脚踩树杈，四肢紧紧地抱住树干，俯身向下调皮地张望，既敢于冒险，又小心谨慎，仿佛与同伴们玩着“捉迷藏”的游戏。简单的画面，将熊猫灵活与调皮的一面生动地展现出来。正如15岁的少年，活泼好动，喜欢挑战。而100元规格的金币设计则相反，一只熊猫安静地坐在一块石头边上，脚下是茂密的草地，旁边是一片竹林，环境清幽，安然恬适。只见它一手放在石头上，一手拿着竹子放在嘴边，神情专注，似乎若有所思，不知是在回顾这15年的成长点滴，还是在畅想自己美好的将来。
