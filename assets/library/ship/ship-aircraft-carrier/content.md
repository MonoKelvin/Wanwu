# 航空母舰

航空母舰（英文：Aircraft Carrier ），是以 舰载机 为主要战斗装备，并为其提供海上活动基地的大型水面战斗舰艇。

船舶类型反映贸易路线、动力革命与海军/民用分工。

## 船体与动力

- **类型**：航空母舰
- **用途**：海上航空兵基地
- **起降**：弹射/滑跃/垂直短距
- **代表**：尼米兹级、福特级等
- **地位**：海军核心战力


## 历史与冷知识

- 帆船到集装箱船体现了贸易全球化与造船工业的演进。
- 相关标签：Aircraft Carrier, 航空母舰, 1917年7月。
