# 最终幻想VII

在开发初期，《最终幻想VII》是为SFC打造的2D游戏。系列创始人坂口博信钟爱像素风，当时团队就JRPG是否转型3D进行多次探讨后，决定在《最终幻想6》完工后，为SFC打造2D的《最终幻想7》，还召开许多立项会议集思广益。但因Square当时开发《时空之轮》需要更多人手，负责《最终幻想7》的小团队被调走，SFC版制作中断。

尽管SFC版开发中断，坂口博信定下的早期大纲是一个侦探故事。在其最初的剧情大纲中，有一个组织试图摧毁纽约市的魔晄炉，名为“乔伊”的侦探调查这伙人，组织成员之一是主角克劳德的原型。这个故事成为游戏开场剧情底稿，只是故事地点从纽约市换成米德加城。

数年前，一家叫神罗的公司（早期叫做神罗工程所）发现了埋藏在地底深处的某种生物遗体（杰诺瓦）和一种叫“魔晄”的能源，并用其作研究，他们将生物体浸入魔晄中生产电力，从而发展成为全球最大垄断公司，魔晄可增加人体能力，但使用过量会让人变成怪物。

世界各地都有采集魔晄能源的魔晄炉，神罗公司背地里拿人体作试验，失败的就被抹消。而魔晄是不可再生的星球的生命能源，过度采集会使星球凋亡。主角们来自反神罗组织雪崩，在与神罗对抗的过程中逐渐了解了魔晄和杰诺瓦的实质，开始为拯救星球而战。

神罗公司为建立掌控星球的庞大产业帝国，抽取星球的生命能源魔晄，导致星球面临危机。反神罗组织 “雪崩” 为拯救星球，对神罗展开反抗行动。克劳德作为曾经的神罗强化战士，加入了雪崩组织。雪崩成员们炸毁了壹号魔晄炉，给神罗以打击，但爆炸也牵连到了第八区。

在第八区，克劳德遇到了卖花女爱丽丝，同时与青梅竹马蒂法重逢。雪崩组织试图炸毁五号魔晄炉时，行动被神罗监控，神罗将壹号魔晄炉爆炸和第七圆盘坠落灾难嫁祸给雪崩，使雪崩陷入孤立无援的境地。爱丽丝为给雪崩争取喘息之机，以自己被囚禁为条件与神罗做交换。

克劳德等人潜入神罗大厦营救爱丽丝，成功救出爱丽丝后，拥有杰诺瓦基因的最强神罗战士萨菲罗斯出现。萨菲罗斯战力超强，克劳德等人只能奔逃。萨菲罗斯有着可怕的计划，而星球的命运守护者菲拉出现，与萨菲罗斯达成共识成为盟友，阻碍人们改变命运。

爱丽丝意识到这是命运的转折点，雪崩成员决定跨越命运之墙。在与萨菲罗斯的激战中，克劳德承载着伙伴的意念劈开了命运，成功打破宿命，众人站在了未知荒野，走上命运的岔路。

打破命运之墙后，克劳德回到现实，雪崩面临被神罗 “围捕” 的命运。扎克斯扶着几乎昏迷的克劳德艰难撤离，还试图营救被神罗控制的同伴。而克劳德一行在脱离魔晄都市米德加后，驾驭着陆行鸟，穿越广袤的草原，开始了一段揭开星球奥秘的冒险之旅。在冒险过程中，他们还将面对神罗公司的追捕、神秘黑袍人携带 “杰诺瓦” 遗体带来的危机以及苏醒的古老守护者 “武器” 等诸多挑战。

故事的主角，蒂法的青梅竹马。性格内向、不爱说话、自卑，这就是克劳德的性格。像很多男孩子一样，年幼的克劳德憧憬强大的力量，因此他将传说中的战士萨菲罗斯视为自己的偶像，梦想有一天能和他一样强，尼布尔海姆的给水塔上，克劳德承诺变强后一定会回来保护蒂法后参军，他的命运因此而转变。

故事的女主角之一，反神罗组织“雪崩”的成员，在米德加7号街经营着一家名为“第七天堂”的酒吧（同时也是“雪崩”的秘密基地）。

克劳德、蒂法、艾瑞丝三人的“三角恋”一直是饱受玩家争议的话题，很明显蒂法对克劳德的感情的确不是“青梅竹马的朋友”那么简单的，蒂法对感情问题比较内敛害羞，从来没有正面表白过自己对克劳德的感情。个性坚强不失温柔，性格沉稳，遇事冷静且考虑周全，对于克劳德的异常表现和错误记忆并没有当场揭穿，而是慢慢观察寻找机会最后终于帮助克劳德找回真正的自我。在克劳德魔晃中毒精神失常时更是时刻陪伴其左右寸步不离。

赛特拉，代代守护着星球并且可以与之交谈，受星球的引导，来到传说中的约束之地，从而获得无上的幸福。这是赛特拉人代代相传的传说，也是他们的宿命。只有她能揭开约束之地的神秘面纱，发动阻止陨石的究极白魔法“神圣”，特殊的身份使艾瑞丝被卷入了无数的纷争和战斗。游戏中盘的突然离去可以说是震撼性的，同时也造就了无数的“复活”谣言，艾瑞丝的人气之高可见一斑。

虽然身背沉重的宿命，身份特殊从小就被神罗四处追讨，但艾瑞丝始终保持着活泼开朗的性格，对待感情问题要比蒂法积极主动得多，这与她文静的外表不太相符。克劳德的已故朋友扎克斯是艾瑞丝的初恋，由于杰诺娃细胞的虚拟记忆，使得艾瑞丝能从他身上找到些许扎克斯的影子，这也是促使她主动接近克劳德的原因之一，但她更在意的始终是真正的克劳德。

数年前的可利尔魔晃炉事故，为掩人耳目神罗杀光了几乎所有的可利尔村民，其中也包括巴雷特深爱的妻子，同时他为保护朋友丹恩而失去了右臂，之后为了复仇他装上了机关臂。

巴雷特脾气暴躁，动不动就暴粗口，这可能与他过去轻信神罗导致家破人亡有关，只有在面对义女玛莲时，他才会显现出温柔的一面。胸前的项链是好友丹恩妻子的遗物，丹恩临终前将它交给了巴雷特保管。

会说人类的语言，拥有不输于人类的智慧的长寿种族，世代守护星球的它们甚至被称 为星球的守护神。赤红十三就是这令人骄傲的圣兽一族的成员，本名为纳纳奇，赤红十三是宝条为方便研究而给它起的代号。虽然实际年龄是48岁，但如果以人类年龄换算的话那么它只有15、6岁大而已，还只是个孩子。

赤红十三幼年时其部族受到基族的攻击而濒临灭亡，不明真相的它始终认为它的父亲在战斗中抛下了整个部族和母亲临阵逃脱，是个懦夫，这是它长年以来的一个心结，而当它查明真相，站在父亲已经石化的遗体旁仰天长啸的场景更是给无数玩家留下了极为深刻的印象。

野村当初在设计这个角色时就想让它充满印第安风格，因此它才有印第安式的发型和装饰，纳纳奇这个名字也纯粹是为了让它更富有印第安风格而起的。

她是乌泰城主高多的女儿，乌泰在和神罗的战争中输掉后被开发成了旅游胜地，尤菲不甘心自己的故乡被神罗统治并认为服从神罗的父亲是个懦夫，她开始周游世界，目的是收集足够多的魔石来复兴自己的故乡。

别看尤菲外表可爱，实际上她是一个非常狡猾且贪心的家伙，她的最终目的就是等克劳德他们的战斗结束后将他们所有的魔石都据为己有。

乘坐着胖莫古利的古怪机器猫，自称占卜机器，其实是可以远距离操纵的 高性能机器人。以要亲眼见证克劳德的命运为借口强制加入队伍，真实身份是神罗派到克劳德队伍中的卧底。
