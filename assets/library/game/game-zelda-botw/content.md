# 塞尔达传说：旷野之息

《塞尔达传说 旷野之息》是任天堂企划制作本部开发、 Monolith Soft 协力开发的动作冒险游戏，为《 塞尔达传说 》主系列的第19作，于2017年3月3日在 Wii U 、 Nintendo Switch 上发售。 [1] 2025年6月5日发售 Switch 2 增强版。 [17] [19] 游戏的故事发生在海拉鲁王国灭亡的100年后，曾经一场大灾难袭击了海拉鲁王国使之灭亡，主角 林克 在地下遗迹苏醒，他追寻着不可思议的声音并开始了冒险之旅。 [2] 游戏于2017年12月8日获得TGA年度游戏、最佳游戏设计和最佳动作冒险游戏；2017年12月23日获 GameSpot 年度最佳游戏；2018年1月5日获EDGE年度最佳游戏；2018年3月24日获得GDC最佳游戏音效奖、最佳游戏设计奖和年度游戏奖 [3] ；2018年3月18日获得SXSW最佳游戏性奖、最佳游戏设计奖和年度最佳游戏。

年度最佳游戏；2018年1月5日获EDGE年度最佳游戏；2018年3月24日获得GDC最佳游戏音效奖、最佳游戏设计奖和年度游戏奖

海拉鲁王国的历史是一部从远古时代开始，便接连蒙受加侬这一灾厄入侵的历史。灾厄加侬是太古时代，在海拉鲁诞生的魔王，他化为怨念的姿态，每隔一段时间便会复活。每到那时，拥有勇者之魂的人就会和继承了女神之血的神圣公主一起为海拉鲁夺回和平。

远古祖先曾制造能被人类所操控、具有野兽外表的四座巨大神兽和具有自我意识与敌人战斗的傀儡——守护者。持有封印之力的王室公主和被驱魔之剑选中的骑士，他们曾在遥远的太古时代，与神兽和防卫者们一同封印加侬。

然而一百年前的某一天，海拉鲁王国的占卜师留下了一个预言说：“给大地带来灾厄的加侬有了开始复活的前兆，可是反抗加侬的力量仍沉眠于大地。”海拉鲁王国的人民也模仿祖先出阵对抗。从海拉鲁选出具有优秀能力的四个人，交由他们操控神兽，并找到被驱魔之剑选中的骑士林克。接着推选公主塞尔达为领导者，其余五人为英杰，巩固力量。在公主和五英杰齐聚一堂之下，原本应该能封印灾厄才对，但是狡猾的加侬却从海拉鲁城堡的地下深处复活，控制了防卫者和四神兽并袭击回来。城镇居民和操控神兽的英杰丧命，林克也为了保护公主而伤重倒下，海拉鲁王国被加侬毁灭了。

但是，幸存下来的公主，在最后时刻释放了封印之力，将被选中的骑士林克送出了城堡，只剩自己挺身对抗着加侬，由于公主一人的力量不足以消灭加侬，最后不得不将自己与加侬一同限制在城堡中。

每通过一个试炼神庙玩家就会获得一个试炼通过证，而集齐四个试炼通过证，便可在散落于海拉鲁大陆各处的海利亚女神雕像处兑换一个心之容器或精力容器，每一个心之容器可为游戏角色增加1颗心，而精力容器可为游戏角色增加1/5圆环的精力槽。

玩家在该作中主要的回复生命值的方式是吃东西，而将生的食材烹饪熟可以提高生命值的回复量或者赋予玩家一些一次性的额外的心，这些一次性的心会以黄色标示。与此同时，某些食物和药剂还可以为玩家在一段时间内赋予一个增益效果，而这些增益效果会在屏幕的左上角显示出来，比较典型的是辣的食物可以为玩家御寒

该作中林克可以驯服野马作为自己的坐骑，玩家可以在马厩里注册自己的马匹，在注册之后可以为马匹取名。任天堂为了表现马匹的动态专门为马作了动作捕捉，因此玩家可以在游戏中操纵马匹以马术舞步动作原地转圈。玩家在马上作战时可以选择让马匹自动奔跑，而玩家可以专注于作战。除了马匹外，玩家还可以骑乘野鹿等，但不能够在马厩注册，而且奔跑速度较慢

玩家扮演的林克将在公主和“四英杰”的帮助和指引下，完成神兽试炼，并解放神兽、解救“四英杰”。其中，塞尔达公主可以与玩家进行“时空对话”，指引玩家逐步返回到100年前林克和公主及“四英杰”所到访过的地方，并回忆起当年的故事。

主要栖息于野外的森林、山区等地方，不过有时候也可在神庙中发现它们的踪迹，它们通常会数只一起出没，或者是和其他敌人一同出现。

外观为有着狮子的头部，以及形似人马身躯构造的兽人，可从嘴巴吐出火球攻击，还能使用手上的刀具进行挥砍，产生大范围或远距离的攻击，由于具有壮硕的体力与强大的攻击力，成为海拉鲁大陆上颇具知名度的猛兽敌人。

可以通过爆炸波对物体造成伤害，或者破坏东西。炸弹有圆形和正方形两种，可以根据情况区分使用。强化后的爆炸波威力上升，至下次使用的时间间隔缩短。

对已停止时间的对象施加冲击，可以积蓄动力。积蓄的力量只要运用恰当，也可以一移动巨大的东西。经过强化，可以停止邪恶生物的时间。

在该作中amiibo是以希卡之石功能的形式使用的。使用amiibo后会在玩家制定的地点召唤出相关的道具，而在游戏内一天的时间里同一个amiibo只能使用一次。卡通林克的amiibo可以召唤出一个箱子和一些鱼，卡通塞尔达可以召唤出一个箱子和一些鲜花以及香草，时之笛林克可以召唤出一个装有五支火箭的箱子和一些生肉，八位数林克可以召唤出一个装有士兵之盾的箱子和一些桶，黄昏公主的狼林克可以召唤出一头有二十颗心的狼，它可以帮助玩家攻击敌人

该作有着动态天气系统，在不同的天气中海拉鲁大地会展现出不同的景象，比如说在雷雨天气时玩家如果穿着金属防具会吸引雷电，如果被雷电劈中会即死，但是如果在雷电劈下前一瞬间将身上的吸引雷电的物品扔向敌人可以将雷电引向敌人。而在下雨时海拉尔的地面会变得湿滑，此时林克进行攀爬会从攀爬的表面滑下来。

在该作中玩家扮演的林克可以使用一个被称之为“静止器”的希卡之石功能，该功能可以停止某个物体的时间流逝，在此期间玩家可以不断击打该物体积蓄动能，在时间重新开始流动后积蓄的动能可以一次全部释放出去，由此可以在衍生出很多新的解谜和战斗的玩法，在右面的动图中就是GameSpot的测评者利用这个功能进行一个长距离的突进，而这只是对这一功能应用的冰山一角。

该作中玩家可以使用大量的不同的武器、防具和服装，并且以此来展开战斗。所有小怪手上的武器都可以被玩家捡起来使用，而由于大部分武器的耐久度并不长因此玩家也会处于一个快速更迭武器的状态。

在以往作为关键道具的部分道具在本作中会以武器的形式出现，比如说回旋镖。玩家在战斗中如果在敌人即将攻击到你时举盾或者闪避可以发动完美防御和完美闪避，在完美防御成功后敌人会出现大硬直，而在完美闪避成功后会进入子弹时间，在子弹时间内可以发动狂风突击。值得一提的是甚至连守护者的激光玩家都可以进行完美防御，在完美防御成功后可以将激光反弹回去。

大部分武器都可以被玩家投掷出去，而同样的大部分武器都可以进行蓄力，在蓄力之后发动的攻击会有更华丽的攻击动作和更高的伤害，比如说点着了火的火炬的蓄力攻击是向四周发射火球。玩家从空中降落时使用弓箭瞄准会消耗耐力进入子弹时间，常见的应用有从马匹上跳起进入子弹时间或者在使用滑翔伞时进入子弹时间。已知的特殊箭支有火箭、冰箭和炸弹箭。箭支命中敌人弱点（比如怪物的头部）时会造成大量伤害，此时会有特殊音效
