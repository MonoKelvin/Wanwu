# 塞尔达传说：旷野之息

《塞尔达传说：旷野之息》是任天堂为 Switch 与 Wii U 开发的开放世界动作冒险游戏，2017 年发售。林克在海拉鲁大陆醒来，探索神庙、收集希卡科技、击败灾厄盖侬，复兴海拉鲁王国。

从读者角度，把「标准书描述」与「真实饲养体验」对照着看，更容易判断自己是否适合该主题：时间投入、预算、空间与家庭成员（老人、幼儿、其他宠物）都会改变答案。以下内容在常识基础上稍作延展，便于形成 3–5 分钟可读完的完整印象。

## 设计亮点

- **物理与化学引擎**：火焰可引燃草地、金属武器导电，鼓励实验性解谜。
- **攀爬与滑翔**：几乎无加载边界的自由探索，塔台解锁地图区域。
- **神庙挑战**：120 座神庙提供独立谜题，不强制线性推进主线。

## 影响

本作重新定义了开放世界 Zelda 的形态，获 TGA 2017 年度最佳游戏，并带动 Switch 主机销量。续作《王国之泪》在此基础上扩展天空与地底区域。

## 冷知识

- 开发团队曾参考京都周边地形进行实地勘测式关卡设计。
- 料理系统可组合数十种食材获得不同增益效果。
- 克洛格种子收集成为玩家间著名的「900 个呀哈哈」梗。
