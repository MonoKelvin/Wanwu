# 集合啦！动物森友会

《集合啦！动物森友会》是由 任天堂 企划制作本部开发并发售的一款模拟经营类游戏，为《 动物森友会 》系列的第七部作品，也是系列首次发售繁体中文版（首次推出简体中文版是 神游科技（中国）有限公司 在2006年6月发售的《 动物森林 》，为《 动物之森 》系列的第一部作品 [1] ）。游戏于2018年9月14日在 任天堂直面会 上首次公布，并于2020年3月20日在 Nintendo Switch 平台全球同步发行 [2-3] 。该作 Switch 2 版于2026年1月15日发售，已经拥有Switch版的玩家可通过升级通行证升至Switch 2版 [45-46] 。 该作与之前的《动物森友会》系列相同，玩家在游戏中可以自由地在系统随机给出的几个地图中选择一个做为自己的居住地。玩家可以在自己居住的地方做任何想要做的事，例如种植花朵、挖掘化石、钓鱼、捉昆虫等。不同的是，游戏新引入了DIY系统，玩家可以用岛上的各种材料制造道具或家具 [4] 。 2020年3月25日，游戏在发售3天后就在日本卖出了188万份，创下了Nintendo Switch平台上首周销量第1的记录（含下载版及套装版），并于同年获Fami通38分评分，入白金殿堂 [5] [22] 。2021年9月，游戏在全球共售出3485万份，为Nintendo Switch游戏销量榜单的第2位 [6] 。2025年9月30日，游戏总销量达到4862万份 [47] 。
