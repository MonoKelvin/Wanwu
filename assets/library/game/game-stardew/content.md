# 星露谷物语

《星露谷物语》是由ConcernedApe单人开发，并与《 星界边境 》的制作者Chucklefish共同参与发行的一款开放式乡村模拟经营类游戏 [1] 。该作于2016年2月27日在PC平台发行，同年12月13日登录PS4平台，12月14日登录Xbox One平台；2017年10月5日登录Switch平台；2018年5月22日登录PS Vita平台，同年10月24日登录iOS平台；2019年3月14日登录Android平台 [4] [7] ；2025年12月26日登录Switch 2平台 [11] 。 在游戏中，玩家继承了爷爷在星露谷留下的老旧农场，并带着爷爷留下的残旧工具和几枚硬币，开始一段全新的生活 [1] 。 2016年4月14日，该作销量达100万份 [8] 。2022年3月，该作销量达2000万份，其中仅PC平台销量就有1300万份 [7] 。2025年7月，该作在Steam平台的评分超越《 传送门2 》，登顶Steam好评榜，成为Steam平台最受欢迎的游戏 [9] ；同年12月，该作入选2025年Steam平台年度热门游玩榜 [10] 。

玩家能把这片杂草丛生的田地变成一个繁荣的家园吗？这很不容易，自从JOJA公司来到了小镇，以前的生活都变了。社区中心，这个曾经是小镇最活跃的地方，如今却是破烂不堪。但山谷里似乎充满了机会，只要玩家肯努力，就有可能把星露谷变得更繁荣

开始游戏后，玩家可以通过捏人的方式创建自己所喜爱的角色。人物共有24种肤色，74种发型，112种衣服，4种裤子，31种配饰。游戏中有生命和能量两种不同的属性

UI位于用户界面右下角能量值的左面。只有角色生命不满或身处矿井的时候，生命值才会显示。在矿山的之外的时候，角色只有在被铁路上的火车撞到，或被秘密森林和史莱姆屋中的史莱姆攻击时，才会消耗生命值。

玩家使用工具、钓鱼、战斗或者是吃了会减少能量的食物时，都会消耗能量。玩家可以通过吃各种食物以及泡温泉来恢复能量。如果玩家能量耗尽，恢复效果会减少一半。吃一个星之果实会让玩家的最大能量值永久的增长34点。

玩家可以通过使用不同的工具进行不同的日常工作，便可积累该领域的技能经验值，进而提升对应的技能等级。更高的技能等级允许玩家解锁该领域内的合成配方，或是选择加成巨大的专业技能来进一步强化人物特长。需要注意的是，通过技能升级解锁的配方需要在该日结束完毕，进行完自动存档后才能获得

村民是游戏中住在鹈鹕镇里及周边的居民。村民有各自的日程表，所以玩家可以根据不同村民的日常安排在不同的地点、时间、日期找到他们。需要注意的是，挡在村民行走路径上的物品可能会永久消失。

武器主要用于与怪物进行战斗。武器可以通过打倒怪物获得、可以在探险家公会购买，可以在矿山中的宝箱中获得，也有极低的概率可以通过钓鱼的宝箱获得

剑属于全面型的武器，挥舞速度很快并攻击延迟很小。剑可以防御反击，可以阻止敌人的伤害并且给予敌人伤害。剑的格挡也可以击退飞行敌人。

匕首在游戏中是攻击最快的武器。可是它们的攻击范围小，并攻击目标只是玩家前面一格，不像剑是个扇形。匕首的第二能力是让玩家可以快速连续攻击敌人。

主要用来敲碎石头和收取放置的设备或家具物品。也可以和锄头一起在非耕种的土地上使用。使用十字镐也能对怪物造成轻微的伤害，与其他武器类似。

用于砍伐树获得木材和树液，也可以用于从大树桩或大树桩中收集硬木。还可以用来收获巨大作物。类似十字镐，它也能对怪物造成轻微的伤害。

玩家可以花费金币和金属锭（或原矿石）在铁匠铺升级玩家的工具，升级需要等待2天时间（如铁匠第三天不在岗，则取货顺延）。在这期间，玩家不可以买东西或要求克林特为玩家打开晶球。工具升级必须按顺序升级。比如铜制工具不能直接升级成铱制工具。

注：工具升级“需要2天完成”的实际含义是在连续2天玩家醒来时工具是不在身上。玩家可以在白天使用工具，然后将其带到铁匠铺（关门前）进行升级。在下一天，玩家将没有工具。在两天后，玩家必须前往铁匠铺获取工具，并在剩余时间内使用。

每个武器或工具只能有一个附魔，玩家不能选择应用哪个附魔，但玩家可以再消耗一个五彩碎片重新附魔。工具会追踪应用在它们身上的前两个附魔，所以当玩家重新应用一个附魔时，之前的两个已出现的附魔不会被重新选择。

游戏中一共有八种农场地图可以选择。每个地图提供一个特殊的农场布局，并提供不同的有利条件和特殊技能加成。开始游戏时只能选择其中一个地图，之后将无法再更改。标准农场提供最大的面积来耕种和畜牧。其他5种类型的农场减少了可耕种面积，但提供了特殊的加成作为补充：

标准农场更有利于耕种；河边农场更有利于钓鱼；森林农场更有利于觅食；山顶农场更有利于采矿；荒野农场更有利于战斗；四角农场更有利于合作模式；海滩农场提供额外物品，鼓励玩家觅食和钓鱼；草原农场生长有特殊的让动物喜爱的耐嚼的蓝草，是适合畜牧的地图。

小镇的西北入口通往巴士站和农场，西南方向通往煤矿森林，那里有玛妮的牧场，莉亚的农舍和法师塔。小镇的南方直接通向沙滩。小镇的北方通往深山，在那里有木匠的商店，矿井，冒险家公会，铁路和采石场。

游戏初期，在2柳巷附近区域，Joja超市和1柳巷屋后的区域，以及社区中心的西边，鹈鹕镇游乐场下边的区域生长有杂草。这些杂草会在冬季时死去，但第二年春天又会重新出现

姜岛是游戏中芬群岛的其中一座岛，也是芬群岛中玩家唯一能进入的岛屿。玩家在修复社区中心或是完成Joja社区发展申请书后，会收到NPC威利的邮件，邀请玩家去鱼店的里屋。前往鱼店的里屋后，玩家可以花费200个硬木、5个电池组和5个铱锭修复里面的旧船，之后便可以花费1000金在那里购买船票坐船前往姜岛。

玩家可以在任何时间乘坐姜岛南部码头的船免费回到星露谷。正常情况下玩家会回到威利的鱼店，当节日时回到农场的路被堵住时，玩家会直接回到巴士站。在月光水母起舞等节日不能前往沙滩的时间段从姜岛回来时，玩家会被传送到鹈鹕镇南边前往沙滩的桥上。

任务是完成固定目标后可以获得奖励。它们记录在任务日志里，点击UI界面底部的小图标或使用快捷键F键进入任务日志。任务以标题形式在日志里列出，每个任务可以展开了解具体描述和需求。点击顶部的X可关闭窗口，在任务日志窗口底部有个按键可以取消任务。接收任务后可以随时完成，并没有时间限制或者过期日期。

一共有两类任务，剧情任务和求助任务。剧情任务会直接发送至农场的信箱，但自动接受的任务是少数的，是在游戏开始时以帮助玩家熟悉游戏

除了贴在布告板上的求助任务以外，还有一些剧情任务随着游戏的进行解锁。玩家收受一个剧情任务包括物品交易的后，玩家可以在日志里取消任务使任务没有完成是没有惩罚的。这些任务不限时间。
