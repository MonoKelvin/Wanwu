游戏中玩家将扮演猎魔人杰洛特进行游戏。通过完成任务和打败怪物可以获得经验点数，在获得一定的经验点数后就能提升等级，并获得能力点数、解锁技能槽与突变诱发物槽来进行人物设定。

技能分为战斗、法印、炼金术、能力，其中又分为核心能力和可学习技能，核心能力作为角色的基础能力，永远处于激发状态并可在战斗、交涉等方面进行使用，是学习其他技能的前提；可学习技能多为被动技能，可以提高基础能力的伤害或改变能力使用方式，每个技能分支下的可学习技能共有20种，需要通过能力点数进行学习和强化，并通过将其放入技能槽激发。未激发的技能不会有任何效果。

每个技能槽旁都有一个菱形槽，用于放置突变诱发物。已激发的突变诱发物能赋予角色数值特殊的加成。不同种类的突发诱变物，可以赋予不同种类的加成。每将一种技能置入同颜色突变诱发物旁的技能槽，就能获得100%的加成。因此，如果突变诱发物旁的三个技能置入了相同颜色的槽，就能获得300%的加成。如果技能与相邻的突变诱发物颜色不同，突变诱发物的加成既不会增加也不会减少。

游戏背景发生在南方的尼弗迦德帝国向北方王国联盟的发起的第三次北境战争，根据玩家在游戏中的不同选择，这场战争会有不同的结束时间和结局。

瑞达尼亚是北方王国中较为富庶的王国，现任国王是拉多维德五世。在游戏中双方为了夺取诺维格瑞区域，在泰莫利亚北部的威伦地区进行了激烈的交锋。

通称史凯利格群岛，地处大陆的西方，由六个大岛屿和十几个零星分布的小岛屿组成。由国王克拉茨·安·克莱特领导，各个家族领袖在各自领地上拥有很大的自治权。他们经常对尼弗迦德帝国的沿岸地区展开劫掠行动，并主张抗争尼弗迦德帝国的入侵。

游戏主人公，利维亚的杰洛特是一名猎魔人，在上古语中，他被称为“格温布雷德”，意为“白狼”。他身手敏捷，但同时又深不可测。在他冷漠寡言的外表下，隐藏着如海洋一般辽阔的善良和幽默感。

像所有猎魔人一样，杰洛特是职业怪物猎手。他拥有超乎常人的力量和速度，杰洛特在猎魔人的青草试炼中展示出不寻常的忍受能力并存活了下来，从而获得了这些能力。

希里全名希里雅·欧菲娜·伊伦·雷安伦，她是一位技艺精湛的猎魔人，好几个皇位的继承人，上古之血的最后传承者，无与伦比的魔法之源。

叶奈法是一位来自亚甸首都温格堡的女术士，她是术士评议会以及其后的女术士集会所成员。她是杰洛特的挚爱，也担当了希瑞养母的角色。

叶奈法与杰洛特的感情始于一场关于迪精的冒险，随着这场冒险，他们二人的命运纠缠在了一起。他们的感情经历了许多风暴，他们仍然互相吸引彼此。

特莉丝·梅莉葛德曾经是弗尔泰斯特国王的前任宫廷顾问，索登山之战的著名英雄、人称山顶第十四人。她拥有聪慧的头脑、温暖的微笑，以及无与伦比的个人魅力。

特莉丝是叶奈法和杰洛特的好友，与后者还有一段坎坷的恋情。特莉丝和希瑞的关系也非常亲近。她在女术士中的影响力跟她对魔法的天份一样，都相当令人瞩目。

“敌人坟头上舞动的白色火焰”，尼弗迦德帝国皇帝，希里的生父。是他策划了对北方王国国王的刺杀，并借此机会发起了第三次北境战争。在他意图统一南北领地的同时，也开始寻找女儿希里以期继承他的皇位。

“铁石心肠”拉多维德五世，瑞达尼亚现任国王。拉多维德的恐怖暴政不仅让他的敌人想要他的命，甚至连他身边的盟友和臣民也有相同的想法。许多人声称他已经疯了，而他对法师和非人类无情的迫害便是其标志之一。

维瑟米尔是全大陆资格最老、最有经验的狩魔猎人，最后的狼学派大师。在故事中维瑟米尔加入了杰洛特的行列，与他一同跨越战争肆虐的泰莫利亚，寻找叶妮法的踪迹。

丹德里恩是著名的吟游诗人、作曲家、歌唱家和作家，也是杰洛特最好的朋友之一。在游戏中，丹德里恩在诺维格瑞城定居，还拥有一家旅馆，并帮助希里寻找修复某种物品的办法。

矮人、佣兵、布伦纳之战生还者、无可救药的乐观主义者、大慈善家、烈酒和黄色歌曲的爱好者、纸牌大师，与杰洛特和丹德里恩是形影不移的好友。

游戏中的武器主要是两把不同用途的剑，这也是猎魔人的象征：一把是对付人类和类人生物用的钢剑，另一把是劈砍怪物用的银剑。敌人生命条的颜色表明玩家应该使用哪种剑进行战斗。在此基础上提供了十字弓进行远程射击，但是伤害比不上剑与法印。
