# 空洞骑士（游戏《空洞骑士》中的最终Boss之一）

空洞骑士（英文名：The Hollow Knight），别称前辈、纯粹容器，是游戏《空洞骑士》中的最终BOSS角色，主题曲包含《被封印的容器》与《纯粹容器》。由 苍白之王 创造的虚空容器，诞生于深渊并使用骨钉作战，其使命为封印瘟疫源头 辐光 。因未能完全消除情感导致感染外泄，最终在黑卵圣殿被玩家操控的小骑士击败。 空洞骑士经苍白之王训练成为最强容器后封印辐光，延缓瘟疫扩散。玩家需击败三位守梦者解除黑卵封印以推进剧情，揭示其源自深渊的诞生过程及容器试炼经历。战斗中拥有冲刺斩、三连斩、扇形飞镖、地刺、圆圈大爆炸及黑触手等招式，在神居以无感染状态的纯粹容器形态登场并展现更高战力。 不同结局将影响瘟疫终结方式，玩家可选择接替封印使命或联合暗影消灭辐光。完成使命的小骑士最终继承"空洞骑士"称谓，其残留情感导致的封印失效机制构成核心剧情线索。黑卵形态被锁链束缚且铠甲残损，双眼呈感染橙色；纯粹容器形态外壳完整双目纯黑，使用未损坏骨钉作战，猎人日志分别记载两种形态特征。

空洞骑士经苍白之王训练成为最强容器后封印辐光，延缓瘟疫扩散。玩家需击败三位守梦者解除黑卵封印以推进剧情，揭示其源自深渊的诞生过程及容器试炼经历。战斗中拥有冲刺斩、三连斩、扇形飞镖、地刺、圆圈大爆炸及黑触手等招式，在神居以无感染状态的纯粹容器形态登场并展现更高战力。

不同结局将影响瘟疫终结方式，玩家可选择接替封印使命或联合暗影消灭辐光。完成使命的小骑士最终继承"空洞骑士"称谓，其残留情感导致的封印失效机制构成核心剧情线索。黑卵形态被锁链束缚且铠甲残损，双眼呈感染橙色；纯粹容器形态外壳完整双目纯黑，使用未损坏骨钉作战，猎人日志分别记载两种形态特征。

玩家在游戏中首次进入黑卵时，空洞骑士正被锁链束缚。此时它仍穿有铠甲，披风比较完整。它的体型明显大于其他容器，身体主要为黑色，头部外壳为锯齿形，外壳上有一道裂痕。它的双目为橙色，视线会随玩家在场景内的移动变化。

在斩断四根锁链后，空洞骑士挣开束缚，铠甲飞开，只剩下残破的披风。它用右臂反手挥舞一根巨大的破损骨钉，左臂则已经残缺，关节较为明显，足部为尖状。在战斗开始时，它会发出一声啸叫，同时屏幕中展示“空洞骑士”字样，随着战斗的进行，它也会发出几次尖啸，实际上这应当都是辐光发出的声音，因为空洞骑士本身是无声的。

在苦痛之路的中，可以见到纯粹容器形象的灵魂图腾，其形象与喷泉广场的空洞骑士雕塑基本一致。而在苦痛之路的尽头和深渊的回忆，以及初次通过骑士万神殿的场景中，玩家可以见到幼年形态的空洞骑士，其形象与小骑士等一般容器相近，只是头部外壳为锯齿状，且其棱角数量和形状与后来的空洞骑士相比也有一定区别。

在神居内，空洞骑士以纯粹容器形态登场。在该形态下，它的双目为黑色，外壳完整，受到攻击不会出现任何感染体征，在战斗中也不会发出任何声音。纯粹容器在战斗前背对玩家，手扶骨钉，身披铠甲，肩甲上有两个穿孔，可能是为了使锁链穿过以束缚它。战斗开始后，它的铠甲会四散飞开，只剩与空洞骑士形态相似的披风。它所使用的骨钉没有损坏痕迹，为纯白色，提骨钉的姿势也与空洞骑士的“拖着”有明显区别。此外，它的左臂完整，但在做一些行动时无法看到。

在苍白之王统治下的圣巢王国，被遗忘的神明辐光正利用着感染逐渐控制和扭曲居民们的思想。为了挽救摇摇欲坠的王国，苍白之王最终选择了以容器封印辐光的方法。

空洞骑士与众多容器一样，是苍白之王与白色夫人的造物，诞生于深渊，被注入虚空的力量，且没有性别。而它是其中唯一被选中的容器，深渊入口处的石碑记录了那场残酷选拔的结果：“我们纯粹的容器已经上升。剩下的只有拒绝和创造它的悔恨。我们不会再进入那个地方。”

在被选中以后，空洞骑士便肩负起封印感染、拯救圣巢的使命。按照苍白之王的描述，作为纯粹容器，它“没有可以思考的心智”、“没有可以屈从的意志”、“没有为苦难哭泣的声音”，因为它没有思想可供辐光渗透，所以是封印感染的最佳选择。

为了保证计划的天衣无缝，苍白之王还安排了三位守梦者在空洞骑士封印感染后再对其进行封印。此三者为教师莫诺蒙、守望者卢瑞恩和野兽赫拉，其中野兽赫拉属于本与苍白之王对立的深邃巢穴势力，为了得到她的协助，苍白之王与她进行了交易，即同她诞下一女——

经过一段时间的养育和训练，空洞骑士达到了容器的最强形态。它能熟练运用巨大的骨钉作战，还能聚集、驱动灵魂和虚空的法术。此时它的力量已可与辐光相抗，并确实一度完成了封印辐光的壮举。后人无从得知那场战斗的具体经过，而它后来残缺的手臂可能就是其激烈过程的证明。

完成封印以后，空洞骑士便被锁链与法术束缚在黑卵圣殿的最深处，三位协助它的守梦者亦随之进入永恒的沉眠，保证它的封印不能被外界破坏。在后来满长的岁月中，空洞骑士坚持对抗着辐光的侵蚀，它也被期望着能永远封印感染并彻底拯救圣巢。

即便是在衰败不堪的圣巢，纪念空洞骑士的雕像仍矗立于泪水之城喷泉广场的中央，旁边还立有三位守梦者的雕像，然而了解它们故事的后者已经少之又少。空洞骑士的雕像高耸而威严，仿佛仍在无声地注视着自己为之奉献过一切的王国。游戏中，阅读纪念碑，能得到这样的描述：“纪念空洞骑士。在那高远的黑色穹顶下。它的牺牲使圣巢永世不衰。”

然而，“纯粹容器”并不纯粹，或许如玩家在小骑士回忆中看到的那样，幼年空洞骑士在深渊短暂的停留和回顾已经暗示了它的命运。亦或许是如白色夫人后来所言，它是由于被灌输的念头产生了“污点”。

尽管它一直抑制着自己对父亲的情感，尽管它有奉献一切以挽救王国的决心，空洞骑士的“污点”仍给了辐光可乘之机，这使它的所有努力与牺牲注定以悲剧收场，而苍白之王对抗感染的各项措施也都成为徒劳。

在游戏的开头，玩家可以看到辐光的力量从空洞骑士体内爆发，使它的外壳破裂，意味着它已经基本被完全感染。而此时的圣巢由于感染未能被真正控制，已经化为一片废墟，苍白之王与他的宫殿也已消失，昔日繁华的泪水之城内遍布游荡的躯壳。可能是受到使命的感召，游戏主角“小骑士”从国王山道一跃而下，来到了这片废墟，在大黄蜂等人的指引下，杀死了三位守梦者，解开了黑卵圣殿的封印，并最终斩断锁链，面对空洞骑士。

空洞骑士的最终命运与玩家扮演的小骑士选择以哪种结局收尾密切相关，而在达成每种结局的过程中，空洞骑士都是玩家必须面对的重要对手。

小骑士如果仅杀死三位守梦者并直接进入黑卵，空洞骑士便会作为最终BOSS与之展开交锋。在一段战斗后，空洞骑士自身的意志与辐光的抗争愈发明显，它开始不断地用骨钉捅向自己体内的感染。最终，小骑士击败了空洞骑士，吸收了它体内封印的辐光，取代它被束缚于黑卵之中，成为新一任空洞骑士。

在小骑士获取虚空之心后，大黄蜂便会在黑卵圣殿外等候，并在小骑士与空洞骑士决战的关键时刻前来帮助。她暂时控制住了空洞骑士，如果此时小骑士连续使用梦之钉以外的攻击方式或一定时间未对它使用梦之钉，便会导致它挣脱，而大黄蜂会因无法承受束缚昏死，战斗继续进行。空洞骑士此时已经虚脱，只能缓慢地移动，每次斩击也都会在挥出第一次攻击后虚弱地倒下，很快就能被小骑士击败。在这种情况下空洞骑士的结局仍然是被小骑士接替，区别在于生死不明的大黄蜂也一同被封印于黑卵中，可能是成为了新一任空洞骑士的守梦者。

如果小骑士抓住大黄蜂控制空洞骑士的时机，用梦之钉进入它的思想内部，就能直面辐光。一番激战后，在同胞们暗影的注视下，虚空控制住了辐光。空洞骑士的暗影此时出现，它撕开辐光的“面部”，使其核心暴露，并最终由小骑士的暗影完成最后一击，使辐光的精华被虚空吞噬殆尽。在终结辐光后，空洞骑士、小骑士与同胞们回归了虚空。

在神居内，小骑士得以在骑士万神殿面对空洞骑士在感染前的形态，即纯粹容器。初次通过骑士万神殿后，玩家能看到一段动画，其情景为：幼年空洞骑士、小骑士和其他容器们相互注视，这时传来辐光的啸叫，容器们纷纷抬头，另一声啸叫随之而来，可能是由暗影之主发出。

小骑士拥有虚空之心后便可进入圣巢万神殿，当它于其中击败纯粹容器形态下的空洞骑士后，空洞骑士会抬头，接着辐光的光芒照下，小骑士进入与无上辐光的决战。最终，虚空粉碎了辐光，感染也随之终结。无论此前小骑士有没有将一朵娇嫩的花送给寻神者以防止虚空的扩散，空洞骑士都会走出黑卵圣殿，并与黑卵外守候的大黄蜂相见，大黄蜂在此出乎意料的情境下摆出了战斗姿态，剧情到此戛然而止。此结局留给了玩家们更多想象的空间。

空洞骑士是唯一被苍白之王选中的容器，被认为是纯粹的。它在经受训练后，拥有熟练使用骨钉，以及驾驭灵魂法术及虚空的能力。它的骨钉技艺相较于残破容器更为致命，后者曾被漫游者艾琳娜认为可能是“一位天赋异禀的骨钉高手”，而它也是少数几位会使用武器进行格挡反击的角色之一。
