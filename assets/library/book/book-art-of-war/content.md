# 孙子兵法

## 概述

> **要点**：春秋时期孙武所撰兵书，约六千字、十三篇，为中国现存最早、影响最深的军事经典，置于《武经七书》之首。

《孙子兵法》又称《孙子》《孙武兵法》，作者传统上认定为春秋齐国孙武。孙武携十三篇谒吴，经伍子胥举荐受重用。全书以「孙子曰」起篇，结构严整、语言简练，把政治置于战争胜负的首位，系统论述战前谋划、用兵原则、地形敌情与火攻间谍等议题。

两千余年来被尊为「兵学圣典」，不仅滋养中国军事思想，亦通过多种译本影响世界战略、管理与商业决策。毛泽东曾推崇其思想；智、信、仁、勇、严亦常被视作军人武德之源。

## 篇章结构

全书十三篇，各篇主旨大致如下：

| 篇名 | 要旨 |
|------|------|
| 计 | 庙算：五事七计，慎战与战前谋划 |
| 作战 | 战争与经济、速战速决 |
| 谋攻 | 全胜思想：攻城为下，攻心为上 |
| 形 / 势 | 攻守之形与奇正之变 |
| 虚实 | 因敌制胜，避实击虚 |
| 军争 | 以迂为直，争利而不乱军 |
| 九变 / 行军 / 地形 / 九地 | 应变、行军与地利 |
| 火攻 / 用间 | 火攻条件与间谍五间 |

名句如「知彼知己，百战不殆」「不战而屈人之兵」「其疾如风，其徐如林」等，已融入日常汉语。

## 思想特点

- **慎战与全胜**：强调战争代价，追求以最小损耗达成政治目的。
- **因敌变化**：反对僵化教条，重视情报与形势判断。
- **朴素辩证**：对立概念（攻守、虚实、奇正）可相互转化。
- **文体影响**：排比、比喻生动，音韵铿锵，兼具哲理与可操作性。

## 版本与流传

1972 年山东临沂银雀山汉墓出土《孙子兵法》竹简，为研究早期文本提供实物依据。后世注本以曹操注最早；宋以来《十一家注孙子》等刻本流传甚广。近代以来英译名 *The Art of War* 使其成为全球阅读最广的中国古籍之一。
