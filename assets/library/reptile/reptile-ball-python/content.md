# 球蟒

球蟒是有鳞目蚺科蟒蛇属爬行动物， [4 ] 又称国王蟒、皇蟒。一般成体长度100厘米，最长可达250厘米。躯体较粗大，颈部较细，显得颈部较突兀。底色淡褐色，具有不规则暗褐色斑纹，尾部呈直条花纹；头部暗褐色，眼部上方有明显的直条纹。因受惊吓后常以头部为中心盘成圆球状而得名。 [5 ] 大多数寿命为20-40岁。 [6 ] 球蟒分布在中非和西非的雨林中。 [7 ] 栖息于草地、热带草原及疏林地带，喜欢躲在其他动物所挖掘的洞穴里。不喜欢强烈的光线，经常在黎明和黄昏的时候活动、觅食。以小型哺乳动物为食，幼蛇则会捕食鸟类。 [6 ] 雌性每次会产4-10枚蛋，然后盘住蛋直到3个月后孵化，期间雌蟒不进食，一直守护着蛋。 [4 ] 球蟒是非洲蟒类中最小的种类。 [8 ] 目前，数量正在逐年递减，已处于濒临灭绝的状态。 [6 ] 被列入《濒危野生动植物种国际公约》（CITES）附录二。 [5 ] [9 ]
