# 红眼树蛙

## 概述

红眼树蛙，是无尾目雨蛙科 红眼蛙属爬行动物 。 雄蛙体型比雌蛙小 ； 背部为鲜艳的亮绿色，身体两侧为紫色，腹表面为白色，脚趾为橘红色，带有白色条纹；脚蹼很大，形状像水杯，能够帮助它在树上爬来爬去进行捕食。红眼树蛙有着猩红且充满活力的双眼，故而得名“红眼树蛙” 。 红眼树蛙主要分布于哥斯达黎加、中美洲和南美洲的热带雨林中。红眼树蛙为完全夜行性群居动物，白天常常蜷缩在树叶上睡觉，夜晚出来觅食，主要以苍蝇、飞蛾、蚂蚱、蟋蟀和其他小青蛙为食 。 繁殖季节雄蛙开始鸣叫，吸引雌蛙。交配时雄蛙抱紧雌蛙，雌蛙会把卵产在水塘旁的树叶的背面，雄蛙同时授精，大约5天，蛙卵会孵化成蝌蚪 。 寿命大约7年 。 红眼树蛙是一种非常罕见的蛙，观赏性极高。红眼树蛙脚趾的末端长有可分泌一种特殊黏液的吸盘，借助这种特有的吸盘可以在光滑的表面自由行走，常被称为“技艺高超的高空工作者” 。

## 特点与细节

众所周知，红眼树蛙是一种非常害羞的动物，它们长着亮红色的眼睛和足部，非常擅长于伪装。在瓜纳卡斯特省阿雷纳火山地区发现红眼树蛙绿色的身体能够帮助红眼树融入树叶背景之中有效地隐藏起来，避开掠食者和猎物的视线。

红眼树蛙有一对大大的、突出的红色眼睛，脚上有橘红色的大脚趾，背部是一片鲜艳的亮绿色，而腹部却是白色的，身体两侧还带着湛蓝色的花边，身上的色彩极其艳丽，看着令人赏心悦目，是极受欢迎的观赏性蛙类。红眼树蛙身上的色彩常被称作“闪光色”，当其跳跃时，这些闪光色会吓走天敌。当它蜷缩在树叶上睡觉时，它把红红的大眼睛藏进脑袋里，腿和脚趾都收拢到身体下面，身上明亮的鲜绿色会变得暗淡，看上去就像是绿色的树叶，天敌就不容易看到它啦!如果有谁打搅它的美梦，它就突然睁开红红的大眼睛，溜圆溜圆地盯着你，然后慢慢伸出那美丽细长的腿，猎食者见到它那红红的大眼睛，都会吓一跳，树蛙就可以借机安全地逃跑了。红眼树蛙的脚蹼大大的，脚趾末端还有能分泌黏液的吸盘，借助这种特有的吸盘，它不仅可以在树上爬来爬去，还能在光滑的物体表面上自由行走红眼树蛙生活在中美洲和南美洲热带雨林中。

红眼树蛙是观赏性极高的品种。猩红且充满活力的红色双眼，背部一片鲜艳的亮绿，再加上身体两侧那无比吸引人的澄蓝色，以及橘红色的脚趾，这些色彩常常被称做“闪光色”。人们认为红眼树蛙在跳跃时，这些闪光色会吓走它们的天敌。红眼树蛙的许多特性使它能在森林里生存下来。它的脚蹼大大的，形状像水杯，能帮助它在树上爬来爬去捉昆虫吃。它的皮肤在夜里会变得暗暗的，这样，它的天敌就看不到它!天亮起来的时候，它的皮肤就会发出绿色的亮光，使它看上去像绿色的树叶。

事实上， 红眼树蛙是一种夜行动物， 在白天休息的时间， 它们的腿部紧缩， 眼睛闭合， 如果不仔细观察很难发现它们的存在。红眼树蛙遍布生活在中美洲雨林， 它们长着特殊的红色眼睛， 带有垂直的窄状瞳孔。它们的身体具有鲜艳的绿色， 还有黄色和蓝色相间的垂直条纹， 蹼足呈现橙色或者黄色， 长有3个眼睑。在白天休息时， 它们会将腿部盘起来闭合眼睛， 在茂密的丛林中很难发现它们。
