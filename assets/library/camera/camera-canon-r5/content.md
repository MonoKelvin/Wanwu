# 佳能 EOS R5

## 8K时代的开创者

2020年7月9日，佳能（中国）有限公司通过线上发布会，正式推出了全画幅专微旗舰机型EOS R5。这款相机一经发布，便在摄影界掀起了轩然大波——它是全球首款支持无裁切8K内录的全画幅数码相机，也是佳能首款搭载机身5轴防抖的机型。在新冠疫情笼罩全球的2020年，EOS R5的发布如同一道曙光，照亮了影像创作的未来。

![佳能EOS R5](https://aka.doubaocdn.com/s/IKel1wSWOM)
*佳能EOS R5全画幅专微旗舰相机*

> [!tip] 命名由来
> EOS R5的命名延续了佳能"5系"的传奇。从1992年的EOS 5胶片相机，到2005年的EOS 5D数码单反，再到2020年的EOS R5专微相机，"5"这个数字承载着佳能最顶级的技术与最纯粹的追求。

## 传奇的延续

### 5系列的荣耀

佳能EOS 5系列相机在摄影史上占据着举足轻重的地位：

- **1992年**：EOS 5发布，成为首款配备眼控对焦的35mm单反相机
- **2000年**：EOS 5D开创了全画幅数码单反的先河
- **2008年**：EOS 5D Mark II将全画幅带入视频时代
- **2012年**：EOS 5D Mark III成为专业摄影的标杆
- **2016年**：EOS 5D Mark IV集大成于一身

> [!quote] 佳能副总裁石井俊幸
> "EOS R5在决定命名时，就考虑了中国用户对相机的叫法以及习惯，最终在全球范围内敲定了这个名字。"

### 划时代的突破

EOS R5不仅继承了5系列的荣耀，更开创了新的时代：

- 全球首款8K无裁切内录全画幅相机
- 首次搭载机身5轴防抖
- 首次实现鸟类眼睛检测对焦
- 首次支持8K RAW视频内录

## 技术规格

### 核心配置

| 项目 | 规格 |
|------|------|
| 传感器 | 约4500万有效像素全画幅CMOS |
| 处理器 | DIGIC X |
| ISO范围 | 100-51200（可扩展至50-102400） |
| 对焦点 | 最多1053个 |
| 连拍速度 | 最高约20张/秒（电子快门） |
| 视频规格 | 8K 30P、4K 120P |
| 防抖 | 机身5轴防抖，最高8级 |

### 传感器与处理器

EOS R5搭载了一枚全新开发的约4500万有效像素全画幅CMOS图像感应器。这枚传感器不仅拥有极高的分辨率，更重要的是具备惊人的数据读出能力，能够支撑全像素下最高约20张/秒的高速连拍，以及8K 30P或4K 120P的高规格视频拍摄。

![4500万像素传感器](https://aka.doubaocdn.com/s/4q5E1wSWOM)
*新开发的约4500万有效像素CMOS图像感应器*

> [!example] 技术突破
> 这枚传感器的数据读出速度是前代产品的4倍，能够在高速连拍和8K视频录制时保持出色的画质表现。

与传感器协同工作的是佳能最新的DIGIC X数字影像处理器。这枚处理器首次应用在旗舰级单反EOS-1D X Mark III上，如今被下放到了EOS R5中。相比EOS R采用的DIGIC 8处理器，DIGIC X的处理性能提升了约6.1倍。

![DIGIC X处理器](https://aka.doubaocdn.com/s/eny81wSWOM)
*高性能DIGIC X数字影像处理器*

DIGIC X不仅能够快速处理庞大的影像数据，还能进行高精细清晰度处理，抑制明暗反差大的边缘锐化易出现白边的现象，提高图像细节表现力。

得益于DIGIC X的强大性能，EOS R5支持以10位色深HEIF格式图像记录。启用相机的HDR PQ拍摄功能后，所拍摄图像会以HEIF格式记录。当HEIF图像输出至兼容HDR模式的显示设备时，可呈现更为丰富的层次与宽广的色域，获得更加接近人眼真实所见的逼真观感。

## 8K视频革命

### 开创8K新时代

EOS R5最引人注目的特性，无疑是其强大的8K视频功能。它是首款支持无裁切8K内录的EOS全画幅数码相机，可进行最高约29.97fps、12位8K RAW短片（8192×4320像素）机内录制。

![8K视频](https://aka.doubaocdn.com/s/UTNJ1wSWOM)
*EOS R5支持8K RAW视频内录*

> [!note] 8K的意义
> 8K视频的分辨率为7680×4320像素，是4K的4倍，是1080P的16倍。这意味着EOS R5能够捕捉到比4K视频多四倍的细节，为后期制作提供了前所未有的灵活性。

### 专业视频功能

除了8K RAW，EOS R5还支持：

- **8K DCI/UHD**：最高29.97P/25P的帧频
- **4K 120P**：超高清升格慢动作
- **4:2:2 10位Canon Log**：专业色彩分级
- **HDR PQ记录模式**：高动态范围视频
- **8K超采样4K**：从8K超采样生成高精细4K视频

对于专业视频创作者来说，EOS R5的这些功能意味着他们可以用一台相机完成从8K素材拍摄到4K成片输出的完整工作流程。8K RAW视频为后期调色、裁切、稳定提供了巨大的空间，而4K 120P则让慢动作拍摄变得更加细腻。

## 智能对焦系统

### 第二代全像素双核CMOS AF II

EOS R5采用了升级后的第二代全像素双核CMOS AF II技术。在"面部+追踪"自动对焦模式下，自动对焦范围扩展到最大约100%画面；对焦区手动选择模式下，自动对焦范围扩展为最大约90%（水平方向）×100%（竖直方向）。

![全像素双核对焦](https://aka.doubaocdn.com/s/wCiq1wSWOM)
*第二代全像素双核CMOS AF II*

相机可根据被摄体位置从最多1053个自动对焦框中自动选择，实现精准的被摄体追踪。自动对焦速度仅为约0.05秒，对焦亮度界限扩展到了EV-6至20，即使在极暗或极亮的环境下也能精准对焦。

### EOS iTR AF X 智能追踪

> [!success] 技术创新
> EOS R5采用了新的EOS iTR AF X（智能追踪与识别自动对焦）技术。基于DIGIC X数字影像处理器的算法和深度学习技术，EOS R5能够实现革命性的自动对焦性能。

EOS iTR AF X的检测能力包括：

1. **人物检测**：检测人物的头部、面部、眼睛，即使被部分遮盖或侧脸也能识别
2. **动物检测**：检测猫、狗的身体、面部、眼睛
3. **鸟类检测**：首次支持鸟类眼睛检测对焦，极大提升"打鸟"成功率

这些功能的实现依赖于佳能积累的大量训练数据和深度学习算法。通过分析数百万张图像，相机学会了识别各种目标特征，实现了在各种复杂场景下的可靠对焦。

## 8级防抖系统

### 手持拍摄的新纪元

EOS R5是佳能首款搭载机身5轴防抖的机型。当搭配使用支持光学防抖的RF镜头时，通过机身和兼容镜头的协同工作，最大可实现8级抖动补偿效果。

![8级防抖](https://aka.doubaocdn.com/s/8k5j2aLZWM)
*机身与镜头协同实现最高8级防抖*

> [!tip] 8级防抖的意义
> 这是目前防抖效率最高的数码相机，让手持低速快门拍摄也能拍出清晰高画质图像。

8级防抖意味着什么？根据CIPA测试标准，使用RF28-70mm F2 L USM镜头，70mm焦距拍摄时，理论上可以使用比安全快门慢8档的快门速度。例如，70mm焦距的安全快门约为1/70秒，8级防抖后可以使用约1/4秒的快门速度手持拍摄。

这一突破性的防抖性能，使得摄影师在以下场景中获得极大便利：

- 室内人像摄影：无需三脚架即可使用低速快门
- 弱光风景：手持拍摄成为可能
- 视频拍摄：提供稳定的画面

### 防抖协同工作原理

EOS R5的8级防抖是机身防抖与镜头防抖协同工作的结果：

1. **机身5轴防抖**：补偿俯仰、摇摆、水平平移、垂直平移、旋转五种抖动
2. **镜头光学防抖**：根据镜头焦距提供针对性防抖
3. **协同控制**：相机协调两者工作，实现最优防抖效果

## RF镜头生态系统

### 全新卡口设计

EOS R5采用佳能RF卡口，这是佳能专为无反相机设计的全新卡口系统。RF卡口拥有54mm的大内径和12个电子触点，为镜头设计提供了更大的自由度，也为机身与镜头之间的高速数据传输提供了保障。

![RF镜头](https://aka.doubaocdn.com/s/yVK81wSWOM)
*佳能RF镜头群*

> [!example] RF卡口优势
> 相比传统的EF卡口，RF卡口的大口径和短后焦距离为镜头设计提供了更多可能性，使得制造更大光圈、更高素质的镜头成为可能。

### 旗舰级RF镜头

RF镜头群的亮点包括：

- **RF 50mm F1.2 L USM**：标准定焦镜皇，极致虚化
- **RF 28-70mm F2 L USM**：全球首款全画幅F2恒定光圈标准变焦镜头
- **RF 70-200mm F2.8 L IS USM**：紧凑型远摄变焦，画质卓越
- **RF 85mm F1.2 L USM**：人像镜皇，梦幻虚化
- **RF 100-500mm F4.5-7.1 L IS USM**：超远摄变焦，打鸟利器

RF镜头不仅光学素质出众，还普遍配备了控制环、防抖系统等实用功能，与EOS R5的协同工作效果极佳。

## 专业操控设计

### 融合经典与现代

操控方面，EOS R5融合了EOS 5D系列与EOS R系列的优势：

- **背部速控转盘**：回归传统单反的操作手感
- **多功能选择拨杆**：快速选择对焦点
- **双卡槽设计**：CFexpress卡+SD卡，兼顾速度与兼容性
- **新型电池LP-E6NH**：更大容量，支持USB充电
- **防尘防水滴结构**：提升恶劣环境下的可靠性

### 高速连拍性能

EOS R5在高速拍摄方面同样表现出色：

- **电子快门**：最高约20张/秒
- **机械快门/电子前帘快门**：最高约12张/秒

使用全新的C-Raw格式，在12fps的机械快门连拍下，R5最高可以连续拍摄大约370张照片，已经超过很多旗舰级体育相机的连拍续航水平。

![高速连拍](https://aka.doubaocdn.com/s/4zQj1wSWOM)
*最高20张/秒的高速连拍*

## 市场反响与行业影响

### 专业评测

EOS R5的发布，标志着佳能正式进入了全画幅无反相机的高端市场。25999元的国行售价，对于一款集4500万像素、8K视频、8级防抖、20fps连拍于一身的旗舰相机来说，极具竞争力。

> [!quote] 专业评价
> - **中关村在线**：9.4分超高分，"2020年到目前为止性能最为强大的一台相机"
> - **ZOL评测**："当年的5D又回来了！"
> - **新浪数码**："划时代的旗舰级微单相机"

### 行业影响

EOS R5的成功，不仅巩固了佳能在专业影像市场的地位，也推动了整个影像行业向8K时代迈进。索尼、尼康等竞争对手随后也纷纷推出了8K视频相机，但EOS R5作为开创者的地位不可动摇。
