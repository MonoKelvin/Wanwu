# 三毛猫（日本文化）

三毛猫（みけねこ，Mike-neko）在日语中指白、橘（茶）、黑三色相间的家猫，尤以短尾、圆脸个体在日本民俗中最具「幸运猫」意象。本条目强调毛色文化与日本短尾、招猫的关联，而非独立国际品种；生物学上三花/玳瑁毛色与 X 染色体色素分布相关，绝大多数为雌猫。

## 背景与历史

江户时代浮世绘、谚语中已有「三毛」吉祥说法；船家、商家豢养三花雌猫辟邪招财。现代遗传学证实：控制橘色与黑色的等位基因位于 X 染色体，雌猫（XX）可出现两色块加白色斑块，雄猫（XY）产生典型三花需额外 X 染色体异常，故罕见。三毛形象与招猫雕塑、日本短尾猫品种标准中的「ミケ」毛色名称一致，构成跨物种符号与真实猫只的叠合。

## 外观特征

- **毛色**：白底上分布橘黄斑与黑色斑，边界可清晰可晕染；与「龟甲」（黑橘无大白）有区别。
- **体型**：多为日本本土中等家猫体型；与短尾基因组合时尾短而弯。
- **眼色**：黄、琥珀、铜色常见；与毛色无严格一一对应。
- **性别**：几乎均为雌猫；若见三花公猫，属遗传例外且常不育。
- **被毛**：以短毛为主，长毛三花亦存在。

## 性格与气质

毛色本身不决定性格，但日本民间对三毛猫有「聪明、稍傲娇、带来好运」的刻板印象。实际个体性格取决于品种背景（如日本短尾则活泼爱叫）与幼猫社会化。饲主调研中，三花雌猫在多国被戏称为「猫中女王」，属趣味标签而非科学结论。

## 饲养与护理

- **领养**：优先绝育雌猫，可减少发情呼叫与乳腺肿瘤风险。
- **健康**：三花与玳瑁共享遗传基础，无额外「三色病」；常规关注肥胖、牙周、肾病。
- **文化尊重**：理解「幸运」为民俗，勿因毛色忽视医疗或遗弃不「标准」斑纹的猫。
- **多猫家庭**：毛色不影响合群，按性格与资源（猫砂盆、食盆数量）管理即可。
- **摄影与版权**：网络「网红三花」形象不能替代真实饲养责任。

## 文化与冷知识

- 日语「三毛」字面意为「三种毛」，与中文「三花」同义。
- 招猫陶瓷常用白身+橘黑斑+黑耳，抽象自三毛短尾雌猫。
- 船猫传统中，雌猫被认为能安抚鼠类与船员情绪，三花因常见而成为象征载体。
- 英语 Calico、Tortoiseshell 与日语三毛/サビ（锈色）分类不完全重合，跨文化比较时需注意术语。
