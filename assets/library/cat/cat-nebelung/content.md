# 内华达猫

内华达猫（Nebelung，亦译尼比龙猫）是 1980 年代在美国由 Cora Cobb 选育确立的长毛品种，外观酷似「长毛版俄罗斯蓝猫」——银蓝被毛、修长体态与翠绿色双眼是其标志。品种名源自德语 *Nebel*（雾），寓意被毛如晨雾般朦胧的银蓝色调。全球数量稀少，在 TICA 等体系有独立登记标准。

## 背景与历史

1984 年，美国科罗拉多州一只名为 Siegfried 的雄性蓝灰色长毛猫与雌性俄罗斯蓝型家猫 Brunhilde 交配，产下长毛银蓝后代。Cobb 夫妇将这一类型系统选育，并于 1987 年向 TICA 提交新品种申请。早期血系以俄罗斯蓝猫与家猫杂交为基础，后引入符合标准的俄罗斯蓝猫以巩固毛色与体型。1990 年代获 TICA 冠军级认可，CFA 等协会登记相对较晚。因种群规模小，正规猫舍通常需预约等候。

## 外观特征

- **体型**：中等、修长而优雅，公猫约 3.5–5 kg，母猫略小；骨量适中，肌肉线条流畅。
- **头型**：略呈楔形，吻部中等长度，大耳直立且耳尖圆润，与俄罗斯蓝猫头型相近。
- **被毛**：半长至长，双层结构，外层银蓝带灰，底层更浅；颈围与尾毛尤其丰沛，触感丝滑。
- **眼色**：幼猫可能呈黄绿，成熟后转为纯绿色；被毛与眼色对比强烈，辨识度高。
- **识别要点**：长毛 + 银蓝单色 + 绿眼；与短毛俄罗斯蓝猫相比，被毛长度与尾羽是主要差异。

## 性格与气质

内华达猫普遍较为含蓄，初识陌生人时可能退避观察，但对认定的家庭成员极为忠诚。多数个体安静、温和，不喜欢过度喧闹的环境。它们常跟随主人在家中移动，享受同处一室但不一定强求被抱。与儿童或其他宠物相处通常良好，前提是给予足够的适应时间。智力中等偏高，可接受轻柔的训练与互动游戏。

## 饲养与护理

- **梳毛**：每周 2–3 次，换毛季增加频率；长毛易打结，重点清理腋下、腹侧与尾根。
- **环境**：偏好稳定、可预测的日常；提供躲藏点与高 perch 有助于缓解应激。
- **健康**：整体健壮，关注常规疫苗、驱虫与口腔护理；与近缘品种类似，可咨询猫舍是否筛查 HCM 等遗传问题。
- **饮食**：优质高蛋白猫粮，控制零食；室内活动量中等，需防肥胖。

## 文化与冷知识

- 品种命名中的 Siegfried 与 Brunhilde 来自瓦格纳歌剧《尼伯龙根的指环》，与德语「雾」之意共同构成 Nebelung 品牌故事。
- 虽外观接近俄罗斯蓝猫，但为独立登记品种，二者不可混称；长毛基因在部分俄罗斯蓝血系中偶发出现，却非同一标准。
- 在欧美猫展中，内华达猫因毛色统一、气质「高冷」而常被形容为「雾中贵族」。
