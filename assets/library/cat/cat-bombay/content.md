# 孟买猫

孟买猫（Bombay）是美国 1950 年代选育的纯黑短毛品种，目标为家养「迷你黑豹」——被毛乌黑至发根、眼色铜金、体态圆润结实。由缅甸猫与美国短毛猫等杂交固定而来，与孟买城无地理关联，名称取自印度城市意象。

从读者角度，把「标准书描述」与「真实饲养体验」对照着看，更容易判断自己是否适合该主题：时间投入、预算、空间与家庭成员（老人、幼儿、其他宠物）都会改变答案。以下内容在常识基础上稍作延展，便于形成 3–5 分钟可读完的完整印象。

## 背景与历史

育种者 Nikki Horner 在美国肯塔基州以深褐色缅甸猫为基础，反复选育加深毛色与眼色对比，1970 年代获得 TICA 认可，CFA 于 2000 年前后纳入冠军级品种。标准强调「黑豹」整体感：被毛无锈色、眼色明亮铜至金色。与英国黑色家猫或普通黑猫不同，孟买猫有固定体型与眼色要求。

时间线与地域背景有助于理解它为何在特定年代走红，以及今日在收藏、实用或文化象征中的位置。


## 外观特征

- **体型**：中等紧凑，公猫可达 5.5 kg，肌肉感圆润而非细长。
- **头型**：圆头圆眼，耳距适中，吻部短，呈现温和「小熊」感而非扁脸波斯型。
- **被毛**：短、贴身、漆光感，纯黑为唯一认可色，幼猫偶有稍浅条纹，成年后应消失。
- **眼色**：铜色至金色，越大越亮越好，与黑毛形成强烈对比。
- **识别要点**：全黑短毛 + 铜金眼 + 圆润体型；与黑豹色孟买混种需看血统文件。

## 性格与气质

孟买猫亲人、安静，有「膝上猫」倾向，喜欢与人同处一室。多数对陌生人友善但需时间熟悉。活动量中等，爱玩互动玩具；与犬、其他猫通常能和平共处。叫声轻柔，适合公寓，但若长期缺乏关注会寻求主人注意。

## 饲养与护理

- **梳毛**：每周 1 次软毛刷即可，黑毛易显皮屑与灰尘，可偶尔用湿巾轻拭。
- **日晒**：长时间强光下黑毛可能泛棕，室内饲养为主并注意防晒。
- **健康**：整体健壮；关注肥胖与牙周病，缅甸系血统偶见颅面缺陷（选购正规猫舍）。
- **饮食**：优质高蛋白猫粮，控制零食，维持理想体重。

## 文化与冷知识

- 孟买猫是唯一以「漆光黑短毛+铜眼」为使命的美洲自创品种，常被称作「客厅黑豹」。
- 在影视造型中，孟买猫常替代黑豹拍摄特写，因性格温顺、体型可控。
- 黑色被毛基因与缅甸猫血统相关，与「非黑」缅甸猫共享部分遗传背景。
