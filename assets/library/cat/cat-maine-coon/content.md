# 缅因猫

缅因猫（Maine Coon）来自美国东北部，是公认体型最大的家猫品种之一，耳簇、长尾、虎斑被毛与友善性格使其享有「温柔的巨人」之称。在北美猫展历史上地位显赫，近年在全球家庭饲养中热度持续上升。

## 背景与历史

品种名源于美国缅因州。民间传说包括「与浣熊杂交」等荒诞故事，并无科学依据。更可信的路径是：殖民时期欧洲船猫与本地长毛猫自然杂交，在严酷气候中保留了大体型、双层被毛与强健骨骼。19 世纪末纽约麦迪逊广场花园猫展中，缅因猫曾被称为「美国本土长毛代表」。20 世纪中叶一度式微，1970 年代后血系恢复，现为 CFA 等协会登记量前列的长毛品种。

## 外观特征

- **体型**：公猫常见 6–11 kg 甚至更高，母猫略小；体长含尾可达一米级，骨量足而不显笨重。
- **头型**：方形吻部、高颧骨、大耳带簇毛，眼色多样，与毛色相关。
- **被毛**：半长至长，外层防水、底层保温，领围与尾毛尤其丰沛；虎斑最常见，也有纯色与其他花纹。
- **识别要点**：耳簇 + 矩形大体格 + 蓬松尾；幼猫成熟慢，完全体貌常需 3–4 年。

## 性格与气质

缅因猫普遍对人友好，不少个体像「跟屁虫」，会跟随主人在家中移动。多数能与其他猫、犬共处，叫声偏 chirp/trill 而非尖锐喵叫。智力与可训练性较好，可学会握手、取物等游戏。活动量中等偏高，需要玩耍与垂直空间，但通常不像东方短毛品种那样持续高叫。

## 饲养与护理

- **梳毛**：每周 2–3 次，换毛季增加；重点清理底绒防止毛球。
- **运动**：猫树、益智喂食器有助于维持体重；肥胖会加剧髋关节压力。
- **健康**：关注肥厚型心肌病（HCM）筛查、髋关节发育不良、脊髓性肌萎缩（部分血系）；正规猫舍应提供基因检测说明。
- **饮食**：大型猫需高质量蛋白，幼猫期勿过快催肥。

## 文化与冷知识

- 2018 年吉尼斯「最长家猫（活体）」纪录曾由缅因猫 Stewie 等个体保持，强化其「巨人猫」公众印象。
- 在北欧神话与民间创作中，常把缅因猫与「森林精灵猫」意象并置，虽为现代营销，却反映其野趣外观。
- 美国多州猫展将缅因猫列为「州猫」文化符号（缅因州官方州猫即为缅因猫）。
