# 三花猫（中华田园三花）

三花猫（Calico）指被毛同时呈现**白、橘（红）、黑**三色的家猫，在中国城乡极为常见，是中华田园猫中最具辨识度的毛色类型之一。民间常将其与好运、招财联系在一起；从遗传学角度看，标准三花几乎均为雌性，雄性三花极为罕见，因而更添神秘色彩。

## 背景与历史

三花并非单一「品种」，而是**毛色表型**——由 X 染色体上的橙色（O）与非橙色（o）等位基因，以及常染色体上的白化斑（S）共同决定。家猫随人类迁徙在欧亚大陆扩散数千年，三色组合在各地自然出现；中国文献与口耳相传中，三花母猫常被视为「旺家」「福猫」。

与**玳瑁猫**（黑橘镶嵌、少白）相比，三花白色面积更大，色块边界更分明。国际猫协将 calico 与 tortoiseshell-and-white 归入相近图案体系；在中国语境下，二者常被统称为「三花系」，但严格区分时：三花 = 白 + 橘 + 黑，玳瑁 = 黑橘混色为主。

## 外观特征

- **毛色**：白底上分布不规则橘（红）斑与黑斑；色块大小、位置每只不同，如同「随机印花」。
- **性别**：约 99.9% 为母猫；雄性三花需 XXY 等异常染色体组合，概率极低。
- **体型**：多为中等体型田园猫，头型、耳距、被毛长短因地域血系差异大。
- **眼色**：常见黄绿、琥珀、铜色；与毛色无固定对应。
- **被毛**：短毛居多，亦见中长毛个体；换毛季需加强梳理。

## 性格与气质

三花猫性格**个体差异极大**，无法像纯种猫那样概括统一标准。民间有「三花猫脾气大」「三花都是小公主」等说法，多为轶事而非科学结论。多数田园三花适应力强、对人亲疏因早期社会化而异；部分个体捕猎本能明显，适合提供互动玩具而非完全放养。

## 饲养与护理

- **绝育**：母猫发情频繁，建议 6 月龄前后绝育，降低乳腺肿瘤与子宫蓄脓风险。
- **营养**：控制体重，避免橘色区域个体易发胖倾向；优质猫粮 + 适量湿粮。
- **梳毛**：换毛季每日梳毛，减少毛球与吞毛症。
- **医疗**：定期驱虫、疫苗；关注泌尿道健康（尤其绝育后母猫）。
- **领养**：优先 TNR（诱捕绝育放归）或正规救助渠道，勿支持无序繁殖。

## 文化与冷知识

- 日本将三花称为「三毛猫（みけねこ）」，同样视其为母猫为主、象征好运。
- BTS 歌曲《Serendipity》以「我是你的三色猫」比喻命中注定的稀有相遇——因雄性三花罕见，被用来表达「独一无二」。
- 三花图案每只独一无二，类似人类指纹，是宠物摄影与表情包的热门题材。
- 与「橘猫」「奶牛猫」一样，三花属于**中国本土家猫文化符号**，常出现在短视频与社区领养帖中。
