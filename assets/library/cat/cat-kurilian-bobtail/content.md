# 千岛短尾猫

千岛短尾猫（Kurilian Bobtail）原产俄罗斯远东与日本之间的千岛群岛（含库页岛南部、择捉岛等），是天然短尾、半长毛、体格壮实的岛屿猫种。野外与家养血统经数十年选育已在 FIFe、TICA 等体系登记，因温顺善猎、对水兴趣的报道而受伴侣猫爱好者关注。

## 背景与历史

千岛群岛气候寒冷、渔猎文化盛行，当地短尾猫长期随人类生活于海岸与村落，捕食鼠类与小型猎物。20 世纪中后期，俄罗斯繁育者系统采集岛屿猫基因，固定尾型与体型，并向欧洲出口。日本一侧亦存在相似表型的岛屿猫，遗传与命名在学术与猫展上时有交叉讨论。今日宠物血系以俄罗斯繁育中心影响最大。

## 外观特征

- **尾型**：短而蓬松，常呈「绒球」或「小刷子」，长度与弯曲度有个体差异，但均为天然短尾基因。
- **被毛**：半长毛、双层毛，冬季毛量丰厚，夏季略薄；毛色含虎斑、玳瑁、纯色等。
- **体型**：中大型，雄猫可达 7 kg，肌肉发达、胸宽、腿中等长度。
- **头脸**：略圆楔形，耳中等偏大，眼杏仁形，表情常被形容为「野猫般机警」。
- **与水**：部分个体喜玩水或旁观流水，但个体差异大，不宜一概而论。

## 性格与气质

千岛短尾猫以温顺、亲人、对儿童友好著称，同时保留猎性——喜欢玩逗猫棒、追逐小球。多猫与犬家庭中常见成功案例。叫声中等，不如暹罗极端「话痨」。适合能提供攀爬与互动时间的家庭，长时间完全关笼不推荐。

## 饲养与护理

- **梳毛**：每周 2–3 次，换毛季加强，防止半长毛打结与毛球。
- **活动**：提供猫树、益智喂食器，满足猎食与攀爬本能。
- **健康**：关注体重与关节；选购时了解父母是否有脊柱、尾椎相关病史。
- **饮食**：优质高蛋白粮，控制零食，大型体格易在绝育后发胖。
- **气候**：耐热一般、耐寒较好，夏季需提供阴凉与饮水。

## 文化与冷知识

- 俄语名称常写作「Курильский бобтейл」，英语圈简称 Kurilian，与「Kuril Islands」地名直接对应。
- 岛屿猫同时生活在俄日争议海域周边，使其成为少數跨两国文化语境的猫种之一。
- 与曼岛猫、日本短尾猫并列为世界著名「天然短尾」案例，尾椎发育机制是猫遗传学比较研究素材。
