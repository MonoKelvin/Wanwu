# 橘猫（中华田园橘）

橘猫并非单一品种，而是家猫中**橘色被毛**（红/奶油色 agouti 或 tabby）的常见表型，在中国民间俗称「大橘」「田园橘」。额部 M 形纹、虎斑或 ticked 底纹与圆胖体型常并存，网络文化中「十橘九胖，一橘顶三虎」等梗使其成为最具辨识度的中华家猫符号之一。

## 背景与历史

橘色由 X 染色体 O 等位基因控制，公猫仅需一个 O 即可呈橘，故**公橘比例显著高于母橘**。中华田园猫数千年自然繁殖，橘色个体因视觉醒目而积累大量民间故事。2010 年代起，社交媒体（如「故宫御猫」「流浪橘猫救助」）推动橘猫成为顶流 IP。生物学上，橘猫可出现在美短、英短、缅因等任意品种，但本条目聚焦中国本土广泛分布的田园橘表型。

## 外观特征

- **体型**：中等到偏大，绝育后易发胖；公猫常见「圆脸 + 肚腩」。
- **被毛**：短毛为主，橘色从浅奶油到深红褐；classic tabby、 mackerel tabby 或 ticked 均常见。
- **额纹**：classic 型几乎必有 M 形额纹，是民间辨认橘虎斑的重要标志。
- **眼色**：金、琥珀、绿色均有；与毛色深浅无固定规则。
- **识别要点**：毛色型而非品种；同一窝可出橘与非橘 sibling。

## 性格与气质

民间印象为「贪吃、亲人、慵懒」，虽属刻板印象，但部分研究提示橘猫可能更 bold、更爱社交——样本偏差与饲养方式亦影响结论。多数田园橘适应力强，对人友善，适合家庭伴侣。活动量个体差异大：幼猫活泼，成年后可偏「 couch potato」。与其他/公猫未绝育时可能 mark 领地、夜间叫春，需绝育与行为管理。

## 饲养与护理

- **体重**：橘猫肥胖率高，需定量喂食、控制碳水，定期称重。
- **绝育**：强烈建议；降低 roaming、打架与生殖系统疾病风险。
- **梳毛**：短毛每周一次即可；换毛季用橡胶刷减少浮毛。
- **健康**：关注泌尿（尤其公猫）、糖尿病与牙结石；领养田园橘建议完成基础体检。

## 文化与冷知识

- 「十橘九胖」与 O 基因、绝育后代谢及主人「喂胖=爱」的文化互动有关，并非基因必然导致肥胖。
- 网络名猫如「故宫鳌拜」「日本车站站长猫玉」等多为橘或橘白，强化公众对橘猫的「福气」联想。
- 橘猫在英语 meme 中称 *ginger cat*，与 redhead 人类刻板印象并置，形成跨文化幽默。
