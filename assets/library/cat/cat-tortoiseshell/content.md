# 玳瑁猫（三花系）

玳瑁猫（Tortoiseshell）指被毛呈**黑（或蓝灰）与橘（红、乳色）镶嵌混色**、斑块边界模糊如玳瑁壳的家猫类型，在中国民间常被视为「镇宅猫」「母猫中的战斗机」。它与**三花猫**同属三花系，但白色面积通常较少甚至几乎没有，更强调两色交织的「泼墨感」。

## 背景与历史

「玳瑁」本指海龟壳的琥珀-棕黑杂色，用以形容猫毛色十分贴切。遗传机制与三花相同：X 染色体橙色基因在雌性两条 X 上随机失活（X-inactivation），形成黑区与橘区马赛克；公猫需 XXY 等异常才可能出现玳瑁表型，故**几乎均为母猫**。

中国城乡玳瑁猫数量众多，属田园家猫自然毛色之一。民间将其与三花一并纳入吉祥叙事；在英语世界称 **Tortie**，与 calico（多白）区分。部分蓝乳玳瑁（灰+奶油色）为稀释色基因版本，在品种猫中亦存在。

## 外观特征

- **毛色**：黑与橘（或蓝与乳）交织，无明显大块纯白；个别「玳瑁白」在胸腹有小面积白。
- **图案**：每根毛可能混色，远观如模糊色块；与清晰分界的 calico 不同。
- **性别**：雌性占绝对多数；雄性玳瑁极罕见且多不育。
- **体型**：田园中等体型为主；被毛短至中长均有。
- **眼色**：黄绿、琥珀、铜色常见，与毛色对比鲜明。

## 性格与气质

英语圈有 **「Tortitude」**（玳瑁脾气）说法，指部分玳瑁猫更 vocal、更强势——科学上**证据不足**，多为轶事与确认偏误。中国网络亦流传「玳瑁母猫脾气大」；实际性格取决于个体、绝育状态与成长环境。绝育后多数玳瑁与家猫无异，可成为温顺伴侣。

## 饲养与护理

- **绝育**：强烈建议母猫绝育，减少发情嚎叫、走失与生殖道疾病。
- **营养**：无特殊配方需求；注意控制体重与饮水。
- **梳毛**：短毛每周梳理；长毛玳瑁需防打结。
- **多猫相处**：毛色不决定社交能力；应遵循慢速引见原则，适用于所有猫。
- **领养**：玳瑁在救助站常见，常被低估——实为极佳伴侣潜力。

## 文化与冷知识

- 日本「招财猫」部分造型借鉴三花/玳瑁意象，象征生意兴隆。
- 雄性玳瑁若出现，常被媒体称为「百万分之一」，实际概率因统计口径而异，但确实稀有。
- 「蓝乳玳瑁」在品种 return 基因下呈灰+淡橙，在品种猫舍中亦受追捧。
- 与三花一样，每只玳瑁花纹独一无二，是「天然艺术品」型家猫。
