# 简州猫（四川四耳猫）

简州猫，又称四川四耳猫，是源自四川省简阳市（古简州）的地方家猫类型，以耳部具有额外「耳屏」或副耳突起而闻名。地方标准与保种倡议已见诸报道，但国际主流猫协尚未广泛单独登记，属于中国本土猫种资源中的区域性代表。

## 背景与历史

简州地处成都平原东部，历史上为蜀中重镇，运河与商贸文化发达。当地民间长期饲养体格结实、捕鼠能力强的家猫；其中耳部多出一对小型附耳的个体被单独留意，逐渐形成「四耳猫」口碑。21 世纪以来，地方政府与爱好者推动地方标准制定、保种场与科普宣传，使其与临清狮猫、狸花猫等一同进入「中国名猫」公众视野。遗传机制上，副耳多为耳廓发育变异的表现，需在繁育中避免夸大畸形倾向。

## 外观特征

- **耳部**：在正常耳廓基部和/或后方可见较小附耳或耳屏状突起，两侧对称者为典型；程度因个体而异。
- **体型**：中大型、骨骼结实、胸宽、四肢有力，适应农家与城郊环境。
- **被毛**：短毛为主，毛色常见虎斑、狸花、黑白、橘色等本土类型。
- **头脸**：圆脸至略方脸，眼圆大而明亮，整体气质「敦实」而非极端楔形。
- **尾**：多为正常长尾，与短尾型日本猫种形成鲜明对比。

## 性格与气质

简州猫普遍被认为适应力强、性情稳定，对家人忠诚，对陌生人有警戒但少极端攻击。捕鼠与警戒本能仍在部分血系中保留，适合有庭院或半户外管理的空间；现代家庭饲养则需从小社会化，避免过度野性。与儿童、犬共处时，若早期接触充分，多数能和睦相处。

## 饲养与护理

- **选购**：认准有记录的血系或正规保种渠道，勿购买人为剪裁、伪造「四耳」的个体。
- **健康**：附耳区域需定期检查清洁，防止污垢堆积引发感染；其余关注常规免疫与驱虫。
- **饮食**：高蛋白、适量运动，防止城郊散养型猫转入室内后肥胖。
- **保种**：支持地方标准下的科学繁育，反对近亲繁殖与唯「耳数」外观的极端选育。
- **法律**：在中国饲养宠物猫需遵守当地养犬管理条例中对猫只管理、免疫的相关要求。

## 文化与冷知识

- 「四耳」在方言故事中常被赋予吉祥、镇宅寓意，与简阳羊肉汤、沱江等地域符号一同出现在地方旅游宣传中。
- 副耳现象在兽医学分类上属于先天性耳廓变异，与某些犬种「额外耳垂」讨论类似，研究价值在于发育生物学而非「多一对听力器官」。
- 简州猫与「简阳」地名绑定，有助于理解地方猫种命名往往源于行政区古称而非现代商业品种名。
