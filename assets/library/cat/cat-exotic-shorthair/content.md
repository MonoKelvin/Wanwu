# 异国短毛猫

异国短毛猫（Exotic Shorthair）常被称为「懒人波斯」——拥有波斯猫的扁脸、圆眼与安静性格，却披着短而密的被毛，梳理负担远低于长毛波斯。1950 年代在美国由波斯猫与美国短毛猫等杂交选育而成，现为全球热门室内伴侣品种。

## 背景与历史

1960 年代，美国育种者希望保留波斯猫温顺气质与扁脸特征，同时缩短被毛、降低护理难度。早期称「Sterling」，后定名 Exotic Shorthair。因需持续与波斯猫杂交以维持头型，血统管理严格。CFA、TICA 均将其与波斯分列为不同品种，但允许波斯血统引入。扁脸程度在近年伦理讨论中备受关注，选购宜倾向「中等扁脸」血系。

## 外观特征

- **体型**：矮胖型，公猫 4–6 kg，骨量粗、胸宽、腿短粗。
- **头型**：圆饼脸，鼻短扁，眼大圆，耳小间距宽，与波斯共享「娃娃脸」结构。
- **被毛**：短、密、蓬松感，触感柔软，毛量视觉上接近长毛但长度短。
- **毛色**：继承波斯色型体系，纯色、虎斑、双色、重点色等广泛。
- **识别要点**：波斯型扁脸 + 短密毛 + 矮胖体型；与英国短毛相比脸更扁、毛更蓬松。

## 性格与气质

异国短毛猫安静、温和，喜欢趴卧与膝上陪伴，活动量低至中等。对家人深情，对陌生人多数谨慎但不具攻击性。叫声轻柔，适合公寓与老年人家庭。不耐长时间孤独，适合家中常有人或双猫作伴。

## 饲养与护理

- **面部**：扁脸结构易有泪痕与皱褶分泌物，每日用温水棉片轻拭眼周与鼻褶。
- **呼吸**：极端扁脸血系可能打鼾、怕热，夏季需空调与通风。
- **梳毛**：每周 2–3 次，换毛季增加，防止毛球。
- **健康**：关注多囊肾（波斯血统相关）、牙周病、角膜问题与肥胖；选购要求父母健康筛查。

## 文化与冷知识

- 在英语圈昵称「Lazy man's Persian」，精准概括其市场定位。
- 卡通形象「加菲猫」原型混合了异国短毛与虎斑特征，推动全球对扁脸短毛猫的认知。
- 猫展上异国短毛与波斯常同组环评审风格相近，但毛长是首要区分标准。
