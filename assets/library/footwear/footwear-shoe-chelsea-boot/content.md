# 切尔西靴

## 概述

切尔西靴是起源于英国维多利亚时代的踝靴，由维多利亚女王御用制鞋师J.Sparkes-Hall于19世纪发明。其采用硫化橡胶松紧带设计取代传统鞋带，具有低跟、圆头、无鞋带等特征，流畅的线条与便捷的侧边设计便利穿脱，最初用于马术活动并逐渐普及为日常鞋款 。 20世纪50年代，切尔西靴经伦敦切尔西区青年群体Chelsea Set的穿着开始流行。60年代披头士乐队频繁穿着此类靴子表演，使其获得"披头士靴"别名并引发风潮。70年代设计师Vivienne Westwood夫妇进一步巩固其时尚地位，经历90年代短暂低谷后于21世纪重新回归潮流视野。当代设计中通过擦色处理工艺强化靴筒线条的利落感，延续冬日的沉稳气场同时注入简洁美学 。
