# 勃肯凉鞋

## 概述

勃肯鞋是德国鞋履品牌 BIRKENSTOCK 生产的鞋类产品，由Johann Adam Birkenstock于1774年创立，最初为皇室定制鞋垫。其核心为符合人体工学的鞋床设计，采用橡树软木、乳胶和黄麻等可再生材质制作，通过均匀分配足部压力减少摩擦，提供行走支撑。经典款式包括凉鞋、包头拖鞋等，鞋面多选用牛皮材质，搭配EVA防滑外底。 品牌早期专注于研发足弓支撑技术，1896年在 法兰克福 开设鞋店销售舒适鞋垫，1902年推出首款流线型足弓鞋垫。1945年研发软木内芯技术后，逐步形成标志性鞋床结构。1964年推出首款现代技术打造的勃肯凉鞋Madrid Model。2020年推出高端品牌线1774，与Jil Sander、Beams等品牌联名。2021年起由L Catterton等集团控股，产品线拓展至袜子、箱包及天然护肤品领域。2023年10月在纽交所上市，2025年推出CARE ESSENTIALS护理系列，含六款足部护理产品，采用纯素配方与可持续包装。同年因未获外观设计版权保护，面临仿制品冲击。
