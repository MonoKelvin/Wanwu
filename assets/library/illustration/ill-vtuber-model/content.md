# Vtuber 立绘

Vtuber 立绘是虚拟主播的 2D/3D 角色原画，供 Live2D 或 Unity 骨骼驱动，需分层部件实现表情与身体动作。

![Vtuber 示意](https://upload.wikimedia.org/wikipedia/commons/9/9c/VTuber.png)

![Live2D](https://upload.wikimedia.org/wikipedia/commons/2/2f/Live2D_logo.svg)

## 作品概述

2016 年起 Kizuna AI 等带动产业；彩虹社、hololive 形成企业化运营。

## 艺术与历史背景

融合插画、动捕与直播互动经济。

## 影响与鉴赏

角色人设与「中之人」关系引发行业伦理讨论。

## 冷知识

- 分层 PSD 需命名规范供 rigging。
- 配件可换装实现营收。
- 2D 与 3D 管线成本差异大。
