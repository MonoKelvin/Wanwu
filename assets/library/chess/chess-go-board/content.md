# 围棋

围棋起源于中国，使用 19×19 路棋盘与黑白棋子，通过围地取胜。规则简洁而变化无穷，东亚文化中长期作为修身与竞技项目，职业段位与 AI 研究均极为活跃。

## 基本规则

- **气与提子**：无气的棋子被提取。
- **劫**：禁止立即回提造成重复局面。
- **数目/数子**：中日韩规则在计分细节上略有差异。
