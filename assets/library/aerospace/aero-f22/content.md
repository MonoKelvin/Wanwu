# F-22 猛禽

## 概述

《F-22猛禽》是由NovaLogic公司开发并于1997年11月30日发行的一款飞行射击类游戏，制作团队将 F-22战斗机 的技术数据应用于游戏设计，以增强真实性 。 游戏中玩家可驾驶 F-22战斗机 执行空战任务。游戏设计了分支任务系统，玩家在战役中的表现将影响后续任务类型选择 。作品强调动作射击要素，支持局域网联网对战功能 。《F-22猛禽》可与同系列的《F-16全方位战隼》及《米格-29支点式战机》在整合性作战空域（IBS）中联合作战 。

《F-22猛禽》、《F-16全方位战隼》与《米格-29支点式战机》可在整合性作战空域（IBS）中一起作战，NovaLogic整合这三套产品后，玩家将可在NovaWorld上的整合性作战空域（IBS）进行更大规模的空战

战机系列在年底将推出《F-16全方位战隼》。这些产品包括《F-22猛禽》、《F-16全方位战隼》和《米格-29支点式战机》，可以在整合性作战空域(IBS)中一起作战。随着F-16全方位战隼及MiG-29支点式战机的相继发行，NovaLogic将会整合这三套产品，玩家将可在NovaWorld上的整合性作战空域(简称IBS)进行更大规模的空战。
