# 哑铃

## 概述

哑铃（英文名dumbbell）是用于增强肌肉力量训练的简易器材，属于举重和力量训练的辅助器械，整体由生铁铸造，中间为铁棒，两端配备实心圆球或可拆卸铁片。按重量分为轻哑铃（1-12磅/0.45-5.44千克）和重哑铃（10、15、30千克），其命名源于使用过程中无碰撞声响的特点 。 该器材可通过更换铁片调节重量，主要用于肌肉复合训练、肌力提升及耐力培养，长期规律练习可改善肌肉线条 。哑铃因便携性和操作灵活成为居家健身常用器械，适合女性、老年人及健身新手进行局部肌肉塑形或多关节训练 。训练方式包括弯举、推举、划船等基础动作，需配合标准姿势和呼吸节奏，避免关节损伤 。轻量级哑铃在年轻女性消费群体中需求显著，近年销量增幅近80% 。医疗领域亦用于慢性阻塞性肺病患者肺康复训练及肩周炎康复治疗，通过特定动作改善肌群功能 。

## 特点与细节

锻炼上肢肌肉是哑铃的拿手好戏，但是哑铃同样可以锻炼腰腹肌肉。如在仰卧起坐的时候在颈后部双手紧握哑铃，可以增加腹肌练习的负荷，提高锻炼效果；进行背屈伸练习时，同样可以利用哑铃增加背部肌肉的负荷，以刺激

的增长；手握哑铃的体侧屈或转体运动，可以锻炼腹内、外斜肌；手握哑铃的直臂前举、侧平举等可以锻炼肩部和胸部肌肉；此外，哑铃还可以锻炼下肢肌肉，如手持哑铃单脚蹲起、双脚蹲跳等等。
