# 悉尼歌剧院

悉尼歌剧院坐落于悉尼港便利朗角，丹麦建筑师约恩·乌松 1957 年竞赛中标，1973 年建成。贝壳状预制混凝土壳体覆盖玻璃幕墙，内含音乐厅、歌剧厅等多功能场馆，为 20 世纪最具标志性的现代建筑之一。

## 设计挑战

- 壳体几何与结构计算曾耗费多年，最终采用球面段拼合方案。
- 2007 年乌松获普利兹克奖，部分归功于此项目。

## 参观

- 临海步道与海港大桥构成经典天际线合影点。
- 内部导览可了解声学设计与施工史。

## 冷知识

- 建造预算超支与政治争议导致乌松中途离开澳大利亚。
- 现为 UNESCO 世界遗产。
