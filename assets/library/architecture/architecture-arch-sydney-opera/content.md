# 悉尼歌剧院

悉尼歌剧院（Sydney Opera House）是位于 澳大利亚 悉尼港 便利朗角的地标性建筑，由 丹麦 建筑师 约恩·乌松 设计。 该建筑始于1955年澳大利亚政府发起的国际设计竞赛，乌松方案从32国233件作品中胜出。1959年3月动工建造，工程因政府更迭导致预算争议，1966年乌松辞职后由本土团队接续建造。1973年由英国女王 伊丽莎白二世 主持揭幕，开幕式首演 普罗科菲耶夫 歌剧《 战争与和平 》。1999年，乌松与悉尼和解并设计了新西柱廊。通过亮灯仪式参与2019年新西兰枪击案悼念、2020年消防员致敬活动。2020年2月起，悉尼歌剧院音乐厅开启改造计划 [11] ，2022年完成音乐厅翻新工程，升级音响系统与舞台设备后重新开放。 [14] 2024年，通过亮灯仪式参与悉尼购物中心持刀袭击事件悼念活动 [12-13] 。建筑群占地1.84公顷，坐落在19米高的花岗岩基座上，包含2700座音乐厅、1550座的歌剧院及420座的小剧场等设施，拥有900个功能房间。其标志性帆状壳顶由105万块瑞典陶瓦铺就，最高壳顶距海面60米，整体造型被称为“船帆屋顶剧院”。 [8] 悉尼歌剧院主要为澳大利亚一流的演出团体和机构以及个人提供演出场地，采用演出场馆与本地艺术团建立长期固定合作关系的运作模式 [15] 。2003年，悉尼歌剧院入选新南威尔士州遗产名录；2005年，悉尼歌剧院入选澳大利亚国家遗产名录，2007年被 联合国教科文组织 列入《 世界遗产名录 》 [16] 。
