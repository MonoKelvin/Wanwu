# 泰姬陵

泰姬陵（ 印地语：ताजमहल 、英语：Taj Mahal ）， 全称泰姬·马哈尔陵 ，位于 印度共和国 北方邦 阿格拉 市郊，地处朱穆纳河南岸、亚穆纳河右岸的莫卧儿花园之中，是 莫卧儿帝国 第五代皇帝 沙贾汗 （ Shah Jahan ）为纪念其王后蒙泰姬·玛哈尔（ Mumtaz Mahal ）而
