# 帕特农神庙

## 古希腊的建筑瑰宝

帕特农神庙（Parthenon）是古希腊雅典卫城的主要建筑，以其完美的比例和精湛的建筑技艺而闻名于世。

![帕特农神庙](https://aka.doubaocdn.com/s/h8Lm5wMPXI)
*古希腊建筑的典范——帕特农神庙*

## 历史起源

### 建造背景
- **建造时间**：公元前447年-前438年
- **建造目的**：供奉雅典守护神雅典娜
- **设计师**：伊克蒂诺斯和卡利克拉特

### 建筑历程
- **公元前447年**：工程启动
- **公元前438年**：主体建筑完工
- **后续装饰**：雕刻装饰持续至公元前432年

## 建筑特色

### 结构设计
- **材料**：白色大理石
- **结构**：多立克柱式神庙
- **设计特点**：完美的视觉矫正技术

### 设计美学
- **黄金比例**：建筑比例符合黄金分割
- **视觉矫正**：柱子微微倾斜，基座呈弧形
- **雕刻艺术**：精美的浮雕和雕塑

## 建筑组成

### 主体建筑
- **柱廊**：46根多立克柱
- **内殿**：供奉雅典娜神像
- **三角楣**：精美的三角形山墙雕刻

### 装饰艺术
- **浮雕带**：描绘神话故事的浮雕
- **雕塑**：菲迪亚斯创作的雅典娜神像
- **壁画**：精美的壁画装饰

## 文化影响

### 世界遗产
- **古希腊象征**：古希腊文明的象征
- **建筑典范**：西方建筑的典范
- **旅游胜地**：每年接待数百万游客

### 艺术影响
- **古典主义**：影响西方建筑数千年
- **文化符号**：民主和文明的象征

---

*帕特农神庙以其完美的比例和精湛的技艺，成为了古希腊建筑艺术的巅峰之作。*
