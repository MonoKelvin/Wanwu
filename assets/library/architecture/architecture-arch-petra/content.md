# 佩特拉（约旦安曼北部的古城、世界文化遗产）

佩特拉（Petra），又称“玫瑰城”或“岩石城”（希腊语译名），《 圣经·旧约 》称其为Sela，位于 约旦 安曼 西南260公里的穆萨山谷，面积26,171公顷，属沙漠气候，因赤红色砂岩在阳光下呈玫瑰色调得名 [7] [10] 。古城始建于公元前5世纪，公元前321年首次载入史册，曾是纳巴泰王国首都，前1世纪在国王阿雷特斯三世统治时期疆域扩展至 大马士革 达到鼎盛，凭借香料贸易商路繁盛，106年被 罗马帝国 吞并，3世纪后因红海贸易兴起逐渐衰落，7世纪阿拉伯军队征服时已成废墟，1812年由瑞士探险家约翰·伯尔克哈特重新发现 [7] [12] 。 古城建筑依山体凿建，融合古希腊与东方传统风格，核心遗迹包括哈兹纳宫、欧翁宫及 古罗马剧场 ，唯一入口为长约1.5公里的西克峡谷 [1] [7] [11] 。遗址内设有特拉石博物馆及遗址博物馆，负责遗迹保护与研究。1985年列入 世界文化遗产 ，2007年当选 世界新七大奇迹 [1] [7] 。2025年5月因暴雨引发洪水致数百名游客疏散 [6] 。
