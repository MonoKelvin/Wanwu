# 鬼妈妈 Coraline

## 概述

Coraline是 定格动画 电影《 鬼妈妈 》（2009）的核心角色。影片讲述女孩Coraline随家人迁入老宅后，通过神秘门进入平行世界，面对“鬼妈妈”的诱惑与威胁，最终突破虚幻陷阱并拯救父母的故事 。 Coraline性格聪慧且充满好奇，在平行世界中遭遇纽扣眼睛的“完美家人”与“诡异居民”。她识破虚幻世界的谎言，联合黑猫破解机关，解救被困灵魂并带领父母逃离，过程中学会“珍视现实情感的真实性” 。 影片通过Coraline的冒险隐喻现实与虚幻的矛盾，强调接纳不完美的重要性。其哥特美学风格与暗黑叙事结合，使角色成长兼具惊悚与温情，Coraline成为定格动画史上兼具勇气与深度的经典少女形象

## 特点与细节

Coraline是个充满好奇心的女孩，她在新家里发现有12个房间，却有13扇门，通过那扇神秘门，她进入了一个与现实平行的世界。在这里Coraline碰见了很多奇怪的人，包括她的“赝品母亲”，后者想将其永远留在这里。而Coraline必须依靠自己的机智、决心和勇气，才能重回现实，拯救自己的家庭。

一个与现实完全平行的魔幻世界。这个被“复制”的虚幻世界从表面上来看只是现实世界的镜子，但与现实里的人和事物不同的是，他们都是居心叵测的“鬼魂”。Coraline在这里邂逅了很多怪人，其中包括她的“鬼妈妈”......导演塞利克介绍，此片具有一定的寓言色彩，暗喻了现实与虚幻之间的矛盾，很有观赏的必要。

哥特式的美学风格贯穿于《鬼妈妈》的始终，从头到尾女主角Coraline都生活在一个雾气氤氲、终日不见阳光、处处神秘诡异的小镇里，连她家的房子都有着一个多世纪的古老历史——老宅几乎是全世界

当Coraline发觉墙角的秘密门道通向一个更加诡异的世界时，影片其实营造出某种温馨的情调——在这里Coraline的一切梦想都得以实现，她的父母甚至都有了复制版，而复制版的父母对Coraline百依百顺。 但当梦醒来之时，这一切又显得那么的可怕。每个父母都是独特的，也许他们有很多缺点，像Coraline的父母关心他们的工作更甚于她，同时也对她有着种种限制性要求，但他们终归是爱她的。我们总对生活和现实有着许多抱怨，梦想另一个世界，另一种人生，殊不知归根到底，美好的东西不一定就在幻想中，也许就在我们身边。

诡异灵巧开头部分是童谣式的和声, 给了片子是童话的暗示，但是旋律曲折诡异，并不阳光，说明这虽然是童话...却注定不是天真浪漫无忧无虑的类型。
