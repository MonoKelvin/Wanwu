# 降临 Arrival

## 概述

《降临》是由 丹尼斯·维伦纽瓦 执导， 特德·蒋 、 埃里克·海瑟尔 编剧， 杰瑞米·雷纳 、 艾米·亚当斯 、 福里斯特·惠特克 、 迈克尔·斯图巴 主演的科幻电影 ，于2016年11月11日在美国上映 ，2017年1月20日在中国内地上映 。 该片根据华裔科幻作家特德·蒋创作的小说集《 你一生的故事 》改编 ，讲述了十二个神秘的不明飞行物降临地球，语言学家露易丝·班克博士以及理论物理学家伊恩·唐纳利受政府派遣前去调查飞行物舱中的外星人的故事 。 2017年2月，该片提名 第70届英国电影学院奖 ·最佳影片 ，同年6月，提名 第42届美国电影电视土星奖 ·最佳科幻电影 。截至2025年12月8日，该片全球票房累计达到约2.13亿美元 。

## 背景与发展

讲述了十二个神秘的不明飞行物降临地球，语言学家露易丝·班克博士以及理论物理学家伊恩·唐纳利受政府派遣前去调查飞行物舱中的外星人的故事

地球的上空突然出现了十二架贝壳状的不明飞行物，悬浮在十二个不同的国家上空，外星人向人类发出了讯号，但人类却并不能够解读。美国军方找到了语言学家露易丝·班克斯

经过数次的接触，露易丝发现外星人使用了一种极为特殊的圆环状的文字，并逐渐了解了其中的奥妙。由于一直无法确定外星人来到地球的真正目的，中国、俄罗斯等四国决定对外星人发起进攻，美方亦决定放弃研究将全部人员撤离，在这个节骨眼上，露易丝终于明白了外星人的意图

语言学家，外星人降临地球后被政府征召与外星人交流获取外星人来地球的目的，在过程中学会了外星语言，获得了预见未来的能力，并且与同事伊恩·唐纳利相恋。她成功构建起交流语言时，逐渐明了自己为何注定成为人类与外星智慧体之间的纽带。


## 特点与细节

理论数学家，物理教授，他将与露易丝·班克一起为政府工作，带领他们的团队破译飞船中的生物所发出的噪音。研究它们制造噪音时是否正在说话，更重要的是外星飞船来到地球有何意图。他加入露易丝·班克的直接接触工作，从而形成了与这两个生物进行沟通的二人组。

军队上校，外星人在地球上降落了十二艘外星飞船，引起了政府广泛的警惕，韦伯想到曾为军方翻译过波斯语机密文件的露易丝，并委托露易丝和伊恩·唐纳利试图与飞船上的来客建立联系，试图向两个黑色七肢桶生物教授英语结构，以询问它们来地球的意图并破译它们的语言。

中情局探员，是露易丝和伊恩他们主要跟政府联络的人。由于事件重大，全球都进入紧急状态，哈尔普恩更加不敢轻易掉心，希望想要尽快得到答案，因此对语言学家和物理学家的研究工作施加了压力。‌‌

剧本创作初期制作组仍然在如何将其从文字转化为影像上挣扎，他们早期尝试构建没有这种冲突的叙事时，感觉非常平淡，感觉没有任何利害关系。这个问题的答案占据了主导地位。没有起伏感。丹尼斯和埃里克·海瑟尔都意识到越是琢磨越觉得没有起伏感，如果外星人真的登陆地球，而公众没有立即得到关于他们目的的答案，大家就会越发恐慌。与语言学家和物理学家相处，听他们如何彼此交谈，最终证明了海瑟尔的方法是正确的，并让海瑟尔形成了推动《降临》故事达到如此高度的关键动态关系。这绝非易事，也花费了大量时间，但他最终交出了答卷，而且完成得非常出色


## 影响与延伸

外星人的“八爪鱼”形象、有如中国水墨画和龙纹的文字呈现形式，以及艾米·亚当斯等人从语言学的角度同外星人进行“第三类接触”的过程，也被营造得十分新鲜真实和身临其境。直到影片结尾揭示真相，观众会发现以往剧情中对于艾米·亚当斯和女儿情感戏的呈现与铺垫，都绝非闲笔，正是来自得知真相之后对这些情感戏的重新审视，内心才能涌起关于人生观、价值观、世界观的感慨与思考。导演丹尼斯·维伦纽瓦相当于通过该片做了一次“如何正确地烧脑”的演示，而《降临》的高潮戏其实并非艾米·亚当斯打电话给中国将军解除危机的那一刻，而是在此后情感真相揭示的那一刻，让观众在影片结束之后，依然会充满感慨、震撼与回味

《降临》总体还是不错的，影像化的外星生物形象，在配乐和环境氛围的配合下，前期呈现出神秘恐怖，但随着剧情的发展，怪异的造型又有些让人心生亲近，对露易丝和外星生物的接触也不感到排斥，这已然是一种成功了。外星飞行器的造型也很与众不同，全景时够震撼，有质感，内部运行的重力理论也很精妙，配上诡异的音效，效果奇佳。

最妙的则是对外星人语言的设计，在所有外星科幻片中外星人要么哼哼啊啊发出怪异音节，要么一口流利英语的境况下，七肢桶的这种涵义丰富的可视化语言非常有想象力，尽管是在原著描写基础上设计的，但真正借用东方水墨的风格，设计出这种契合东方哲学周而复始哲学思想，又非常有美感的语言形象，依然是影片的成功之处
