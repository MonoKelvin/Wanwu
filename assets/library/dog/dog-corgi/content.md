# 柯基犬

## 英国皇室的宠儿

柯基犬（Welsh Corgi）是一种源自英国威尔士的小型犬种，以其短小的身材、长长的身体和蓬松的屁股而闻名。它们不仅是英国女王伊丽莎白二世的最爱，更是全球最受欢迎的犬种之一。

## 历史起源

### 古老血脉

柯基犬是英国最古老的犬种之一，其历史可以追溯到至少公元1107年。

根据历史记载，柯基犬是由维京人带到威尔士的瑞典瓦歇犬与当地牧羊犬交配而成。威尔士语中"cor"意为"矮"，"gi"意为"狗"，所以"corgi"就是"矮狗"的意思。

### 两种柯基

柯基犬实际上分为两个独立的品种：

| 特征 | 彭布罗克威尔士柯基犬 | 卡迪根威尔士柯基犬 |
|------|----------------------|--------------------|
| **尾巴** | 短或断尾 | 长尾巴 |
| **起源** | 维京/佛兰德血统 | 凯尔特血统 |
| **体型** | 稍小 | 稍大 |
| **耳朵** | 较小，较圆 | 较大，较尖 |

### 英国皇室最爱

伊丽莎白二世女王从7岁开始养柯基犬，在位期间总共养过30多只柯基犬。第一只柯基犬名叫Dookie，是她父亲乔治六世送给她的礼物。

## 传说与神话

### 精灵坐骑的传说

威尔士传说中，柯基犬曾经是精灵的坐骑。它们肩胛骨上的深色斑纹被称为"精灵马鞍"，据说这是精灵骑手留下的印记。

### 农场守护者

在成为皇室宠物之前，柯基犬是威尔士农场的得力助手。它们被称为"heelers"，通过咬牛的后脚跟来驱赶牛群。它们矮小的身材使它们能够在牛腿间灵活穿梭，避免被踢伤。

## 性格特点

### 核心气质

柯基犬虽然体型小巧，但性格却像大型犬：

- **勇敢自信**：小型犬的身体，大型犬的灵魂
- **聪明伶俐**：智商排名第11
- **活泼好动**：精力充沛
- **忠诚粘人**：对主人非常依赖

### 行为特征

柯基犬以其独特的"微笑"表情和蓬松的屁股而闻名。它们的耳朵竖立，表情警觉，总是给人一种精力充沛的感觉。

## 外貌特征

### 体型结构

| 项目 | 标准 |
|------|------|
| 肩高 | 25-30cm |
| 体重 | 雄犬 10-14kg，雌犬 9-13kg |
| 寿命 | 12-15年 |

### 独特特征

柯基犬有一个独特的"空心背"结构，这是它们能够灵活奔跑和跳跃的关键。它们的后腿肌肉发达，使它们能够做出惊人的跳跃动作。

## 冷知识与趣闻

### 1. 女王的柯基犬外交

伊丽莎白二世女王经常用柯基犬进行"宠物外交"。她会将柯基犬作为礼物送给其他国家的领导人，包括美国总统和日本天皇。

### 2. 柯基犬的"sploot"姿势

柯基犬有一种独特的放松姿势叫做"sploot"，即四肢伸展趴在地上，肚子贴地。这个姿势不仅可爱，还有助于它们在炎热天气里散热。

### 3. 不要给柯基犬剃毛

柯基犬的双层被毛有自我调节温度的功能。剃毛会破坏它们的毛发结构，导致皮肤问题和晒伤。

### 4. 柯基犬的叫声

柯基犬虽然体型小，但叫声非常洪亮，被称为"大嗓门小矮子"。

## 健康护理

### 常见健康问题

- **椎间盘疾病**：长身短腿容易导致脊椎问题
- **肥胖**：容易发胖，需要控制饮食
- **髋关节发育不良**：遗传性关节问题

柯基犬每天需要大量的运动和智力刺激。如果长时间独处或缺乏运动，可能会变得具有破坏性。

## 饲养指南

### 训练要点

柯基犬聪明但固执，训练需要耐心和一致性。它们有自己的想法，不会盲目服从命令。

### 运动需求

柯基犬虽然腿短，但运动需求很高。每天至少需要1-2小时的户外活动。

---
