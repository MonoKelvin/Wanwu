# 苏格兰牧羊犬

苏格兰牧羊犬（Rough Collie，常见称「苏牧」）是苏格兰高地培育的长毛牧羊犬，因小说与影视《灵犬莱西》（Lassie）成为全球最知名的犬种形象之一。它们聪慧、敏感、外形飘逸，适合有精力陪伴与梳理的家庭。

## 背景与历史

苏格兰高地牧民需要能在崎岖地形看守羊群、抵御掠食者的犬只。粗糙被毛型与光滑被毛型（Smooth Collie）同源，19 世纪维多利亚女王喜爱苏牧，推动其进入上流社会。1860 年代伯明翰犬展后名声大噪。1940 年代 Lassie 电影与电视剧使品种家喻户晓。今日 rough 与 smooth 在多数犬会分登记，标准相近。

## 外观特征

- **体型**：肩高约 56–61 cm，体重雄犬 20–34 kg，雌犬略轻，体态略矩形。
- **头**：吻长直，颅平坦；眼杏仁状，多为深褐，「白苏牧」为颜色变体非 albino。
- **耳**：小，半立或耳尖折向前。
- **被毛**：粗毛型外层毛直、底绒厚，颈胸形成华丽鬃毛；毛色常见貂、三色、蓝陨石。
- **尾**：长毛下垂，末端微上翘。

## 性格与气质

苏牧极其聪明，学习快，对主人情绪敏感，严厉惩罚易造成怯懦。对儿童友善，有轻微护卫性，对陌生人可能吠叫警戒。需要「有事可做」——服从、敏捷、牧羊或嗅闻皆可。不适合长期关院或忽视，否则出现强迫症吠叫。

## 饲养与护理

- **运动**：每日 45–90 分钟散步与脑力游戏。
- **梳毛**：每周 3–4 次，换毛季每日，防耳后、后腿毛结块。
- **健康**：关注 Collie Eye Anomaly（CEA）、MDR1 药物敏感基因、髋关节发育不良；用药前建议 MDR1 检测。
- **环境**：适应寒冷气候，夏季剪毛不宜过短以免晒伤。

## 文化与冷知识

- Lassie 角色由雌犬扮演，因雄犬毛量更丰、换角方便。
- MDR1 突变使部分苏牧对伊维菌素等药物不耐，是品种重要医学常识。
- Smooth Collie 与 Rough 仅被毛不同，性格相近却曝光度低得多。
