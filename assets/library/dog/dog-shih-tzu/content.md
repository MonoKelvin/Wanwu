# 西施犬

西施犬（Shih Tzu）名字意为「狮子犬」，源自西藏寺院犬与北京宫廷犬的杂交选育，在清代皇室中地位显赫。长而华丽的被毛、扁平颅面与友善性格，使其成为经典玩赏伴侣，亦常见于美容比赛与家庭饲养。




据信与拉萨犬、北京犬等共享亚洲玩赏犬血统。20 世纪初因政治变动流散至欧洲，英国定名 Shih Tzu 并制定标准。二战后在美国普及。现代展示强调圆润头型、大眼与流畅被毛，宠物家养则常见「幼犬剪」或「运动剪」以降低护理量。


- **体型**：小型，肩高约 20–28 cm，体重 4–8 kg，略长于身高。
- **头型**：圆颅、短吻、下颚宽；耳被毛遮盖，眼大圆而暗。
- **被毛**：长而直的双层毛，可触地；任何毛色均可接受。
- **尾**：尾毛丰盛，高举于背侧。
- **识别要点**：狮子般长毛 + 扁面 + 友善表情；与拉萨犬相比吻更短、体型更方。


外向、亲人，喜欢坐在人膝上或沙发上。对孩童与访客通常友好，吠叫多为招呼而非攻击。独立玩赏犬传统使其能独处片刻，但仍需每日互动。不适合长时间户外高温运动。


- **美容**：每日梳毛防结，定期洗澡与眼部清洁；扁面需注意泪痕与呼吸道。
- **运动**：短距离散步加室内游戏即可。
- **训练**：耐心一致，如厕训练可能略慢于工作犬。
- **健康**：关注眼球突出损伤、呼吸困难（短吻综合征）、椎间盘病。


- 英文名源自粤语「狮子」音译，与佛教护法狮像造型有关。
- 宫廷图画与瓷器上常见类似造型，是研究古代玩赏犬的重要线索。
- 「西施」作为美容模板，衍生出多种顶部结造型（top knot）文化。

## 背景与历史

品种标准在 19–20 世纪随犬展与育种俱乐部逐步固化：体型、被毛与性格描述被写入书面规范，便于裁判与繁殖者统一语言。今日选购幼犬时，建议向繁殖者索要**健康筛查**（髋关节、眼科、心脏等）与父母 temperament 记录，而不仅看毛色或体型。

## 外观特征

请结合本地犬舍与标准书核对肩高、被毛与头型；不同血系在体型与运动需求上可能有差异。

识别品种时除毛色外，应综合**头型比例、步态与尾巴**是否符合标准；混血犬可能具备相似毛色但骨架比例不同。拍摄对比图或参加犬展观摩，比单看网络图片更易建立正确印象。

## 性格与气质

幼犬期 **3–16 周** 的社会化窗口尤为关键：接触不同的人、声音、地面材质与温和犬只，能显著降低成年后的恐惧或攻击倾向。成犬性格也受训练方式影响，**正向强化**通常比恐吓式训练更利于长期稳定。

## 饲养与护理

定期兽医体检、疫苗与驱虫应按当地法规与生活方式制定；室内犬注意牙齿与体重，工作线犬种则需防止运动不足引发的破坏行为。与品种相关的遗传病筛查可咨询品种俱乐部推荐项目。

## 文化与冷知识

- 选购前建议实地接触幼犬父母，观察环境与社会化情况。

许多名犬在影视、广告与历史上留下印记，但媒体形象（如「超聪明」「一定护主」）可能掩盖个体差异。领养或购买前，建议实地相处并确认与自家生活节奏匹配。
