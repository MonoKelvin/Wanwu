# 排球少年！！

《排球少年！！》（ハイキュー!!）是古馆春一所创作的少年漫画，2012 年至 2020 年在《周刊少年Jump》连载。身材矮小的日向翔阳憧憬「小巨人」球员，升入乌野高中加入濒临解散的排球部，与天才二传手影山飞雄从对立到搭档，冲击全国大赛。

![《排球少年！！》漫画第一卷封面（维基共享资源）](https://upload.wikimedia.org/wikipedia/en/9/92/Haiky%C5%AB_Volume_1_Cover.png)


## 背景与历史

古馆春一 2012–2020 年连载排球漫画，Production I.G 2014 年动画。日向翔阳以身高劣势依靠速度与跳跃，与影山飞雄组成「怪人快攻」。县大赛、全国大赛篇章节奏紧凑。

## 故事梗概

乌野高中重建「怪人组合」，通过练习赛、县预赛、春高全国逐步成长。青城、白鸟泽、音驹、稻荷崎等强校构成阶梯式对手。作品以战术图解、团队配合与心理描写见长，几乎每场关键分都有角色内心独白。

## 主要角色

- **日向翔阳**：快攻与跳跃，后排防守弱点。
- **影山飞雄**：精确托球，「王者」气质。
- **月岛萤 / 山口忠**：理性拦网与后排发球。
- **泽村大地 / 菅原孝支**：队长与副攻调度。
- **及川彻 / 牛岛若利**：对手学校的明星二传与王牌。

## 制作与影响

Production I.G 动画 2014 年开播至 2020 年第四季，剧场版众多。带动日本排球人口讨论，现实球员亦曾致敬作品。


## 影响与口碑

在日本掀起排球热潮，学校社团人数上升；第四季东京奥运年播出时机加成。与《灌篮高手》《黑子的篮球》并列推荐运动动画。

## 冷知识

- 古馆春一曾是排球部成员，战术有专业基础。
- 「快攻」视觉化分镜影响体育漫画表现。
- 结局乌野未在全国夺冠，强调过程成长。


## 延伸观赏

作品在院线、流媒体与收藏市场上持续获得新观众回顾；官方设定集、分镜集与创作者访谈适合在观赏正片后延伸阅读，有助于理解美术、配乐与叙事取舍。


## 队伍与成长

乌野高中排球队从「飞不起的乌鸦」重建，日向与影山从对立到组成快攻组合。音驹、青叶城西等对手各有风格，比赛分镜强调战术而非超能力。第三季白鸟泽战以单场比赛贯穿，展示心理、体力与团队协作的层层递进。


## 收藏与讨论

影迷常从摄影构图、配乐动机、改编与原著差异等角度重看；蓝光收藏版、评论音轨与幕后纪录片能补充正片未展开的制作背景。学术与自媒体分析亦有助于把作品放回当时的电影工业或动画产业语境中理解。
