# 死亡笔记（2004年大场鸫原作，小畑健作画的漫画）

《死亡笔记》是由日本漫画家 大场鸫 （原作）与 小畑健 （作画）合作创作的悬疑推理漫画。该漫画短篇原型于2003年8月在《周刊少年Jump》刊载并获得读者投票第一位，正式连载版于2003年12月至2006年5月在该杂志刊登，共108话，单行本发行13卷。该漫画在中国大陆因涉及“神秘主义”“死亡”等元素，被2007年「 扫黄打非 」专项行动列为恐怖类非法出版物，并于2015年6月被 文化和旅游部 列入网络动漫产品「黑名单」 [4] 。2020年小畑健宣布推出《死亡笔记》新短篇集《DEATH NOTE短编集》 [7] 。故事围绕高中生 夜神月 意外获得死神 琉克 遗落的死亡笔记展开，通过书写人名即可致死的规则，他以「 基拉 」身份制裁罪犯并试图构建新世界秩序，与天才侦探L展开心理博弈和逻辑对决。作品包含死亡笔记使用规则、死神之眼交易等设定 [3] 。中国大陆市场上曾出现根据漫画改编的衍生笔记本，部分学生将其用于书写师生姓名，2007年专项行动共查缴图书5912册、音像制品1364盘 [6] 。该作品改编的电视动画于2006年在日本上映，共37集 [5] 。2016年日本推出真人电影续作《 死亡笔记 Light up the NEW world 》，讲述夜神月与L对决十年后六本笔记重现人间的故事 [8] ，2017年 Netflix 制作美版真人电影并宣布将拍摄续集 [9] 。

死神琉克拥有可以写下名字就可以杀人的死神笔记本，却将它遗失在人间。天才高中生夜神月捡到这本笔记本，本以为是恶作剧的他，在笔记本上写下了电视中播放的罪犯的名字，确认了笔记本能力的真实性。厌倦腐朽世界的夜神月使用笔记本的力量，开始杀死犯罪者，以此希望建立一个没有犯罪的美好理想世界，而自己成为“新世界的神”。

一本死神的笔记本，拉开了天才与天才之间卓越巅峰的头脑战争的序幕。本已对人间失去兴趣的硫克，在旁观惊心动魄的斗智过程中感叹“人类，真有趣。

死神琉克因厌倦了死神界的生活，故意将一本死亡笔记丢弃在人间。同样厌倦人间生活的高中生夜神月在拾到死亡笔记后，因试验对象死去以及遇上寻找笔记的琉克而相信死亡笔记是真的，决心以笔记的力量铲除罪犯，建立一个没有犯罪的世界，并成为新世界的神。而后以“KIRA”的身份制裁罪犯，造成大量罪犯不分国籍死于心脏麻痹，成了ICPO会上议题。神秘名侦探L断言这是大量杀人事件，宣布已开始着手调查，要求ICPO协助，并在日本与警察成立调查总部。在L调查下，确定KIRA就住在关东并向其发出挑战。受到L的挑战，夜神月认为L将会成为他创造新世界路上的绊脚石，而接受其挑战并与之展开对决。
