# 境·界（久保带人创作的漫画）

最初考虑的死神的服装是西服，男性是黑色西装，女性是天主教会学校的制服。但当时在电影中已有穿黑色西装的死神。自古以来的死神是套着黑色的布的骷髅，对此进行改变而成的穿黑色西服的死神也已经有过了。那么还没有出现过的死神的样子到底是怎么样的呢。于是就想到黑色的和服。那么与和服搭配，所以武器就选择了刀。

是个看似暴力、单薄，实质上善良、勇敢、爱护家庭的少年，并且拥有能看见灵的体质。家里有一个开诊所的老爸和两个性格正常的妹妹夏梨和游子，一护每天七点必须按时回家，否则老爸便会使用“身体语言教训”的家规教训一护。吵闹的父子，懂事的妹妹以及与其他普通人并无大异的普通生活，直到女主角死神

被他一脚踢到墙角并满脸惊疑地望着他问“你能看见我？”时漫画的序幕才这样被正式地揭开。 从此黑崎一护身边所有的事物都产生了翻天覆地的变化。

斩魄刀是死神所使用的佩刀。其形状和能力，是以死神自身的灵魂为原型筑成的。死神通过知晓赋予自己的斩魄刀的名字，通过与之进行心灵的对话而得到力量。与死神一同降生，和死神一同消亡，这就是斩魄刀。

斩魄刀的创造者是零番队的“刀神”二枚屋王悦，他独自一人打造了所有的斩魄刀（除了黑崎一护的斩魄刀，二枚屋王悦在527话出，在同集中打造了一护的斩魄刀）。

舍弃咏唱即是不用经过言灵程序就能使出鬼道，在战斗时机上有更好的发挥。不过会因应个人鬼道素质和鬼道的难度减弱一定鬼道的威力。后述咏唱就是专门施在以舍弃咏唱方式的鬼道上，加强舍弃咏唱后的鬼道威力。另外，也有非死神的灵体透过自修习得死神才会的鬼道，甚至是研发自己特有的鬼道（例如志波空鹤和志波岩鹫）。

浦原喜助在寻找突破死神的强度界线时所找到的答案“死神的虚化”，借由将这两个相反的存在分解，再重新组合得以追求更高一层的境界，或是将虚的魂魄灌进另外一个灵魂，借由破坏魂魄的界线，尝试将目标升华至更高层次的灵魂。但虚化意谓著灵魂将与自身化成的虚融合在一起，两者将成为无法分离的存在，而死神最终将必须面临被自身化成的虚所吞噬的命运。

随着虚化的恶化，原本的魂魄会变得和虚相同，成为失去理智的怪物。当魂魄间的界线、魂魄外界和外界的界线被破坏时，灵体就会自我消灭，称为“魂魄自杀”。

然而少数人借由“内在虚化抗争”成功战胜体内的虚之后，才将得以随心所欲的应用这股“虚化”的力量，而能够按照自己意识使用这股力量的人则被称作“假面”，因为当他们使用这股力量时脸上将戴上虚的面具，而散发出的灵压充满了不吉的压迫感。更高等的假面，甚至还能施展破面的归刃技能。目前到达此境界的只有黑崎一护、东仙要以及假面军势的众成员。其中东仙要以及假面军势则是经由蓝染以“崩玉”的力量一手创造的，只有黑崎一护是透过自然的“虚化”变化的。

死神的虚化。假面的军势（简称假面）共有8人。其实，“仮面の军势”在很久以前是尸魂界的队长级人物，但其后在一次任务中中了蓝染的圈套，被当作实验对象而虚化，幸好浦原及时出手相救，但中央四十六室相信蓝染，便被下令消灭他们。最后由夜一解救他们到现世避难，浦原、夜一、铁斋和假面也因此流落到现世。而后假面军团再次回到死神阵营，帮助护庭十三队对抗蓝染与十刃。

失去力量的死神紧急时所利用的肉体。供需要长期留于现世的死神使用，穿上义骸后的死神可以让一般人看见他们。一般的义骸可以吸收大气里的灵子使死神的灵力逐渐恢复，但浦原特制给露琪亚的义骸却会逐渐削弱使用者的灵力，最后将使用者由死神变成凡人。为了确保死神的灵体与义骸协调，会需要特定的接着剂来固定灵魂与义骸，可是灵魂和义骸太过同调的话，会让死神在离开义骸的同时感到相当痛苦。后来浦原喜助甚至发明了可快速充气使用的义骸。此外，死神如果没有在现世穿戴义骸，甚至无法食用现世的食物和饮料。

至于浦原喜助给一心使用的义骸，则是用人类魂魄所制成，能将进驻的死神魂魄包覆为完全的人类，成为介于死神和人类间的存在，但是在使用义骸的期间将会无法使用死神之力，也无法看见虚。

登场的角色来自不同的世界，可分作不同的生命体，每种生命体身内所含的“灵子”，浓度不同，因而力量也有所变化。已出现的灵体，如下列：生活在现世的人类、灭却师、魂魄、虚、死神、拥有死神化力量的虚—破面，以及拥有虚化力量的死神—假面，每位角色可按不同的类别来划分。

“仍在世的人类”当中，已登场的角色全部居住于空座町。此地因为灵子浓度极高，居住在此的人亦容易拥有灵力，曾经被蓝染惣右介定为侵略目标，这里的角色性命危在旦夕。

某些人类拥有高度灵力，可以目击“整”、“虚”等魂魄，其中部分会协助死神的工作如消灭虚、引导整等（这种人类被称为代理死神），但有些人类会威胁到世界的安宁（如巴温特，完现术者）。

某些具备虚的力量的特殊人类灵能力者。他们的亲人在他们出生前被虚袭击，虚的力量因此留在母亲身上，并且在他们出生时传到他们的身上，拥有这种能力的人，就被称作完现术者，他们的特有能力则被称为“完现术”（Fullbring）。

在现世死去的人类灵魂，一旦为了某种原因依恋现世而不肯前行至安置魂魄的尸魂界，便成为孤魂野鬼，四处飘荡。“魂魄”通常不会危及人类，并会与能够看见他们的人对话。然而，“魂魄”由于内心深沉的渴望，假如未被及时指引往尸魂界，便可能成为恶灵，亦即“虚”。

灵子浓度极高的生命体，拥有超过人类的能力，也可以与虚战斗。因为身体内灵子浓度高，同时也是虚最喜爱的食物。死神肩负着保卫人类，引导及净化“虚”、使之回到尸魂界的职责，维持两边世界的平衡，以免世界失稳而灭亡；并且是尸魂界的主宰。

在现世的魂魄，过了一段时期，内心的欲望便会腐蚀身体，使之幻化成“虚”，亦即恶灵。虚会四处吞噬其他仍在生的人的灵魂，尤其是生前所爱的人。曾出现的虚，包括井上织姬的大哥、杀害小孩柴田勇一的连环杀手、六年前杀害黑崎一护母亲的虚等。一般在现世所见的虚，仍保有与能够看见他们的人对话的能力，亦即理智并非全灭。由于多数虚内心深处仍有良知，只要有其他灵体的协助，他们仍可过渡至尸魂界，等待转世。然而，大多数虚居于虚圏，力量及体型均远较流落现世的虚庞大，与死神及人类为敌，通称大虚。

大部分虚为了弥补自己的灵力，会吞噬人类魂魄。而有的虚则会吞噬同类，吞噬了几百只后，成为没有特殊能力、体型巨大、智力低下、能够施放

有些特别强大的虚在融合为基力安之后仍能保持原有的记忆、能力和智力，成为融合后基力安身体的主宰，这些少数的有意识体基力安不断吞噬其他的基力安生存，慢慢进化为智力、意识、能力和体型较为完全的中级大虚—亚丘卡斯。
