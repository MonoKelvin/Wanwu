# 三星 One UI

One UI 是三星为其 Galaxy 手机与平板定制的 Android 界面层，2018 年随 Galaxy S9 系列与 Android 9 正式取代旧版 Samsung Experience。它强调**单手可用性**、大圆角卡片与与 Google Material 并行又差异化的视觉识别。

## 背景与历史

One UI 1.0 将可交互区域下移至屏幕下半区，顶部留给浏览内容；2.x 强化深色模式与 One UI Home 启动器；3.x 对齐 Android 11 对话气泡与隐私指示器；4.x 引入 Material You 取色融合；5.x/6.x 持续优化折叠屏（Fold/Flip）多窗口与 S Pen 手势。Good Lock 等官方模块允许高级用户深度定制。

## UX 原则与产品影响

- **Reachability**：关键按钮与 Tab 靠近拇指热区，优化大屏手机单手操作。
- **视觉层次**：Settings 等列表使用 grouped card，减少全屏白板的疲劳感。
- **Samsung 生态**：与 Galaxy Watch、Buds、DeX 桌面模式的 UI 连贯性。
- **定制与预装**：主题商店、图标包与 Always-On Display 是差异化卖点。

One UI 证明 OEM 可在 AOSP + Material 基础上建立鲜明品牌，同时需跟进 Google 隐私与安全 UI 规范（如权限对话框）。

## 冷知识

- One UI 名称中的「One」寓意「一个连贯体验」，覆盖手机、平板、手表与电视。
- 三星 Knox 安全 UI（工作资料夹、安全文件夹）有独立图标与锁屏交互。
- Galaxy Z Fold 的「Flex Mode」半折状态有专门的应用布局指南，One UI 内置分屏预设。
